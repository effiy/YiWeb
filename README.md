# YiWeb - 个人导航与信息聚合平台

## 项目简介

YiWeb 是一个现代化的个人导航与信息聚合平台，采用模块化设计，为用户提供便捷的快捷方式管理、新闻资讯查看等功能。

## 项目架构

### 核心模块

1. **首页模块 (index.html)**

   - 项目入口页面
   - 提供导航功能
   - 量子美学主题设计

2. **快捷方式模块 (shortcuts/)**

   - 功能：管理和展示常用网站快捷方式
   - 文件结构：
     - `index.html` - 快捷方式页面
     - `shortcuts.js` - 主要逻辑
     - `js/` - 模块化 JavaScript 文件
     - `styles/` - 样式文件
     - `data/` - 快捷方式数据

3. **新闻模块 (news/)**

   - 功能：新闻资讯聚合与展示
   - 文件结构：
     - `index.html` - 新闻页面
     - `main.js` - 主要逻辑
     - `api/` - API 接口
     - `components/` - 组件
     - `store/` - 状态管理
     - `styles/` - 样式文件
     - `utils/` - 工具函数

4. **公共样式系统 (css/)**

   - 功能：提供全局样式和变量系统
   - 文件结构：
     - `index.css` - 主样式文件
     - `variables.css` - CSS 变量系统
     - `global.css` - 全局样式
     - `modules/` - 模块化样式
     - `welcome/` - 欢迎页面样式

5. **共享组件系统 (shared/)**
   - 功能：提供可复用的组件和工具
   - 文件结构：
     - `components/` - 公共组件
     - `utils/` - 工具函数
     - `config/` - 配置文件

## 技术特性

### 量子美学主题

- 采用量子美学设计理念
- 支持深色主题
- 丰富的渐变色和动画效果
- 现代化的玻璃拟态设计

### 模块化架构

- 采用模块化设计，各功能独立
- 组件化开发，提高代码复用性
- 统一的样式系统和变量管理

### 响应式设计

- 支持移动端、平板和桌面端
- 弹性布局适配不同屏幕尺寸
- 优化的触摸交互体验

## 样式系统

### CSS 变量系统

项目使用统一的 CSS 变量系统，所有颜色、间距、字体等都通过 CSS 变量进行管理：

- **颜色系统**：量子美学主题色彩
- **间距系统**：统一的间距标准
- **字体系统**：现代化字体搭配
- **动画系统**：流畅的过渡效果

### 样式架构

```
css/
├── index.css          # 主样式文件
├── variables.css      # CSS变量系统
├── global.css         # 全局样式
├── modules/           # 模块化样式
└── welcome/           # 欢迎页面样式
    ├── welcome.css    # 欢迎页面主样式文件
    ├── welcome-core.css    # 核心样式
    ├── feature-cards.css   # 功能卡片样式
    ├── themes.css          # 主题样式
    ├── animations.css      # 动画效果
    └── responsive.css      # 响应式设计
```

## 开发指南

### 文件结构

- `index.html` - 项目主页
- `css/` - 样式文件
- `js/` - JavaScript 文件
- `shared/` - 共享组件和工具
- `shortcuts/` - 快捷方式功能
- `news/` - 新闻功能

### 样式规范

1. 使用 CSS 变量进行主题配置
2. 遵循模块化样式架构
3. 保持响应式设计原则
4. 使用语义化的类名

### 组件开发

1. 将可复用的组件放在 `shared/components/` 中
2. 使用统一的组件接口设计
3. 保持组件的独立性和可测试性
