# YiWeb - 智能新闻助手

## 项目概述

YiWeb 是一个基于 Vue3 的智能新闻助手系统，提供新闻浏览、搜索、分类等功能。项目采用模块化架构，具有良好的可维护性和扩展性。

## 项目结构

```
YiWeb/
├── apis/                    # API接口层
│   ├── helper/             # API辅助工具
│   ├── modules/            # API模块
│   └── index.js           # API入口
├── css/                    # 全局样式
├── plugins/                # 全局插件
│   ├── calendar/          # 日历组件
│   └── searchHeader/      # 搜索头部组件
├── utils/                  # 工具函数
├── views/                  # 页面视图
│   ├── news/             # 新闻页面
│   ├── shortcuts/        # 快捷键页面
│   └── welcome/          # 欢迎页面
└── index.html            # 主页面
```

## 核心功能

### 新闻页面 (views/news/)

- **新闻浏览**: 按日期浏览新闻内容
- **搜索功能**: 支持关键词搜索新闻
- **分类筛选**: 按新闻分类进行筛选
- **标签统计**: 显示新闻标签统计信息
- **日历导航**: 通过日历快速切换日期
- **响应式设计**: 支持移动端和桌面端

### 快捷键页面 (views/shortcuts/)

- **快捷键展示**: 分类展示各种快捷键
- **搜索功能**: 支持快捷键搜索
- **分类过滤**: 按分类筛选快捷键
- **复制功能**: 一键复制快捷键到剪贴板

### 欢迎页面 (views/welcome/)

- **功能卡片**: 展示系统主要功能
- **快速导航**: 提供快速访问入口

## 技术架构

### 前端框架

- **Vue 3**: 使用 Composition API
- **模块化设计**: 采用 hooks 模式组织代码
- **响应式数据**: 使用 Vue 的响应式系统

### 代码组织模式

每个页面采用以下结构：

```
views/[page]/
├── data/
│   ├── mock/             # 模拟数据
│   └── store/            # 状态管理
├── hooks/
│   ├── useComputed.js    # 计算属性
│   └── useMethods.js     # 业务方法
├── plugins/              # 页面级插件
├── index.js             # 页面入口
├── index.html           # 页面模板
└── index.css            # 页面样式
```

### 状态管理

- **createStore()**: 创建响应式状态
- **useComputed()**: 组合计算属性
- **useMethods()**: 组合业务方法

## 使用方法

### 本地开发

1. 克隆项目到本地
2. 使用本地服务器运行项目
3. 访问 `index.html` 开始使用

### 功能使用

1. **新闻浏览**: 在新闻页面查看每日新闻
2. **搜索新闻**: 使用搜索框搜索特定内容
3. **日期导航**: 使用日历组件切换日期
4. **分类筛选**: 使用分类按钮筛选新闻
5. **快捷键**: 在快捷键页面查看各种快捷键
