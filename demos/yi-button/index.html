📋 YiButton API 属性

属性类型默认值说明
variantString''primary / secondary / outline / ghost / danger / accent
sizeString''small / large
loadingBooleanfalse加载状态，同时禁用点击
disabledBooleanfalse禁用状态
blockBooleanfalse块级宽度 100%
iconString''图标语义名（iconMap 查找）
iconPositionString'left'图标位置: left / right
asString'button'渲染为 button 或 a 标签
