📋 示例模态框

这是一个 {{modalSize}} 尺寸的 YiModal 演示。

支持 ESC 关闭、点击遮罩关闭、自定义 header/footer 显示。

✅ 特性：三段式布局 · 键盘无障碍 · body 滚动锁定

📋 API 属性 Props

属性类型默认值说明
visibleBooleanfalse控制显示/隐藏
titleString''模态框标题
sizeString'medium'尺寸: small / medium / large
showHeaderBooleantrue是否显示头部栏
showFooterBooleanfalse是否显示底部操作栏
showCloseBooleantrue是否显示关闭按钮
maskClosableBooleantrue点击遮罩是否关闭
escClosableBooleantrue按 ESC 是否关闭

📡 事件 Emits

close 关闭事件 · open 打开事件（自动锁定 body 滚动）
