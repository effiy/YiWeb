🚀 如何新增一个 Demo？

# 1. 在 demos/ 下新建目录
mkdir -p demos/my-demo/

# 2. 创建自包含的 index.html（零构建，直接可用）
touch demos/my-demo/index.html

# 3. 在本页 demo-grid 中添加卡片
#    复制现有卡片结构，修改 icon / name / path / desc / tags

# 4. 通过 http-server 或项目自带的静态服务访问
http://127.0.0.1:8081/demos/

💡 所有第三方库通过 CDN ESM 加载，无需 npm install。Demo 页面完全自包含，可直接在浏览器中打开。
