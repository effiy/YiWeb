## 功能概述

mermaid 图工具栏功能为每个渲染完成的 mermaid 图表右上角添加一个浮动工具栏，包含**全屏显示**和**下载 PNG 图片**两个按钮。用户可通过全屏按钮将图表放大到全屏模式以便仔细阅读复杂图表，也可通过下载按钮将图表保存为 PNG 图片文件供分享或存档使用。

该功能基于已有的 Plugin 架构（`ToolbarPlugin`、`FullscreenPlugin`、`DownloadPlugin`）实现，三个插件协同工作，通过 `afterRender` 钩子在图表渲染完成后自动注入工具栏 UI。

**核心价值**
- 🎯 **即时全屏**：复杂架构图、时序图可一键全屏查看，无需缩放页面
- ⚡ **一键导出**：将 mermaid 图表导出为 PNG 图片，便于文档分享和存档
- 📖 **非侵入式**：工具栏仅在鼠标悬停时显示，不影响正常阅读体验

## 验收标准

### P0 - 必须通过

- [ ] **工具栏注入**：所有 mermaid 图渲染后，右上角均有工具栏，包含全屏和下载两个按钮
- [ ] **全屏功能**：点击全屏按钮后，图表以模态框全屏展示，图表内容完整清晰
- [ ] **退出全屏**：全屏模式下支持点击关闭按钮或按 ESC 键退出
- [ ] **下载 PNG**：点击下载按钮后，浏览器触发下载，文件格式为 PNG，文件内容为当前图表

### P1 - 应该通过

- [ ] **悬停显示**：工具栏在鼠标悬停时显示、离开时隐藏，不遮挡图表内容
- [ ] **样式适配**：工具栏按钮样式与项目暗色主题（CSS 变量）保持一致
- [ ] **全屏缩放**：全屏模式下图表自适应屏幕尺寸，支持滚动查看超大图表
- [ ] **下载文件名**：下载文件名包含时间戳，如 `mermaid-diagram-1714780800000.png`

### P2 - 可以有

- [ ] **全屏图表缩放**：全屏模式下支持鼠标滚轮缩放图表
- [ ] **工具提示**：按钮 hover 时显示工具提示（tooltip）文字
- [ ] **下载分辨率**：下载时支持 2x 高清分辨率输出

## 功能详情

### 工具栏 UI（ToolbarPlugin）

**功能说明**：在 mermaid 图渲染完成后，通过 `ToolbarPlugin.afterRender` 钩子在 `.mermaid-diagram-wrapper` 的右上角注入工具栏 DOM 元素。工具栏使用绝对定位浮于图表右上角，包含全屏按钮和下载按钮的插槽。

**价值**：统一管理工具栏 UI 结构和样式，使 `FullscreenPlugin` 和 `DownloadPlugin` 只需关注功能逻辑，无需各自创建 UI 容器。

**解决的痛点**：避免多个插件各自注入独立 UI 导致布局混乱；统一 CSS 变量样式确保主题一致性。

---

### 全屏显示（FullscreenPlugin）

**功能说明**：在工具栏的全屏按钮槽位注入全屏按钮，点击后创建全屏模态遮罩层，将 SVG 图表内容克隆到模态层中展示。支持点击遮罩背景、点击关闭按钮、按 ESC 键三种方式退出全屏。

**价值**：复杂的 mermaid 图（如含多层节点的 `graph TB` 或 `sequenceDiagram`）在文档中因容器宽度限制会被缩小显示，全屏模式可让用户清晰查看所有细节。

**解决的痛点**：文档阅读场景中，受限于内容区宽度，复杂图表往往难以辨认；全屏模态框利用整个视口展示图表，彻底解决阅读障碍。

---

### 下载 PNG（DownloadPlugin）

**功能说明**：在工具栏的下载按钮槽位注入下载按钮，点击后将 diagram 容器中的 SVG 元素序列化为字符串，通过 Canvas 转换为 PNG Blob，最终触发浏览器下载。文件名格式：`mermaid-diagram-{timestamp}.png`。

**价值**：用户可将图表保存为图片文件，用于 PPT、邮件、Wiki 等场景分享，无需截图。

**解决的痛点**：SVG 格式不被所有工具支持，PNG 是通用图片格式；手动截图操作繁琐且清晰度受屏幕分辨率影响，程序化导出可保证高质量输出。

## 使用场景示例

#### 📋 场景一：查看复杂架构图

> **背景**：文档中有一张包含多个子图和节点的系统架构图，在文档页面中显示较小，节点文字难以辨认。
>
> **操作**：用户将鼠标悬停到图表上，右上角出现工具栏；点击全屏按钮（⛶图标），图表以全屏模态框展示。
>
> **结果**：用户可在全屏模态框中清晰阅读所有节点和连线文字，按 ESC 键或点击右上角 ✕ 退出全屏，回到文档。

---

#### 🎨 场景二：导出图表到 PPT

> **背景**：用户需要将文档中的 mermaid 时序图插入到 PowerPoint 演示文稿中。
>
> **操作**：用户将鼠标悬停到图表上；点击下载按钮（⬇图标），浏览器自动弹出保存对话框。
>
> **结果**：用户获得一个清晰的 PNG 图片文件，可直接插入到 PPT 或其他文档中使用。
