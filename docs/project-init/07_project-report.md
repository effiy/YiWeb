## 关键发现

### 架构特点

1. **零构建架构**：浏览器直接加载 ES Modules，无需打包
2. **CDN 组件库**：共享组件统一放在 `cdn/components/`
3. **插件化 Markdown**：支持 9 种渲染插件扩展
4. **分层状态管理**：Store 按功能拆分为多个 ops 模块
