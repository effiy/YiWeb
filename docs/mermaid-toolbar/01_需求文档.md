## 功能概述

本功能为 `src/views/aicr/index.html` 页面中的 Markdown 预览里的 Mermaid 图表添加浮动工具栏，为每个渲染完成的图表提供**全屏查看**和**下载 PNG 图片**的能力。

### 核心价值

- 🎯 **提升阅读体验**: 复杂图表可以全屏查看，避免空间限制
- ⚡ **便捷分享**: 一键下载为 PNG 图片，方便分享和存档
- 📖 **非侵入式**: 工具栏仅在悬停时显示，不干扰正常阅读

## 验收标准

### P0 - 必须通过

- [ ] 每个渲染的 Mermaid 图表右上角都有工具栏
- [ ] 工具栏默认隐藏，鼠标悬停图表区域时显示
- [ ] 工具栏包含「全屏」按钮，点击后图表全屏显示
- [ ] 工具栏包含「下载」按钮，点击后下载 PNG 图片
- [ ] 全屏模式支持 ESC 键退出
- [ ] 工具栏不影响图表的正常渲染和显示

### P1 - 应该通过

- [ ] 按钮有悬停视觉效果
- [ ] 全屏模式有清晰的关闭按钮
- [ ] 下载的 PNG 图片质量良好（2x 缩放）
- [ ] 工具栏样式与页面整体风格一致

### P2 - 可以有

- [ ] 右键点击下载按钮可下载 SVG 格式
- [ ] 支持复制图表代码到剪贴板
- [ ] 支持重新渲染图表

## 功能详情

### 工具栏功能

工具栏为每个 Mermaid 图表提供：

1. **全屏查看**
   - 按钮图标: ⛶
   - 功能: 将图表放大到全屏查看
   - 交互: 点击按钮 → 显示全屏遮罩 → ESC/点击关闭/点击遮罩关闭

2. **下载图片**
   - 按钮图标: 📷
   - 功能: 将图表下载为 PNG 图片
   - 交互: 左键点击 → 下载 PNG; 右键点击 → 下载 SVG

### 显示逻辑

- 默认状态: 工具栏透明 (opacity: 0)，不可点击
- 悬停状态: 工具栏不透明 (opacity: 1)，可点击
- 动画: 0.2s 淡入淡出过渡
- 位置: 图表右上角 8px 偏移

### 技术约束

- 不修改 mermaid 核心渲染逻辑
- 通过插件方式扩展功能
- 确保与现有 Markdown 渲染流程兼容
- 无外部依赖新增（使用浏览器原生 API）
