## 功能概述

为 src/views/aicr/index.html 页面中的 Mermaid 图表添加浮动工具栏，提供全屏查看和下载 PNG 图片功能，让用户能够更方便地阅读复杂图表并进行分享或存档。

🎯 **核心价值 1**: 全屏查看功能让复杂图表更易阅读
⚡ **核心价值 2**: PNG 下载功能支持图表的分享和存档
📖 **核心价值 3**: 浮动工具栏设计简洁易用，不干扰正常阅读

## 功能详情

### 浮动工具栏

工具栏定位在每个 Mermaid 图表的右上角，默认隐藏，仅在鼠标悬停时显示，提供不干扰正常阅读的便捷操作入口。

### 全屏查看功能

点击全屏按钮后，图表将在全屏覆盖层中显示，背景变暗，突出图表内容，便于仔细阅读复杂的流程图、架构图等。可通过点击关闭按钮或 ESC 键退出全屏模式。

### PNG 下载功能

点击下载按钮可将当前 Mermaid 图表导出为 PNG 格式图片，支持保存到本地，便于分享、嵌入文档或存档。右键点击下载按钮可下载 SVG 格式图片。

## 使用场景示例

### 📋 场景 1: 阅读复杂架构图

**背景**: 用户在代码审查页面查看包含复杂架构图的文档

**操作**:
1. 用户将鼠标移动到 Mermaid 图表上
2. 右上角显示浮动工具栏
3. 点击全屏按钮（⛶）
4. 图表在全屏覆盖层中放大显示
5. 仔细阅读完毕后，点击右上角关闭按钮或按 ESC 键退出

**结果**: 用户能够清晰地查看图表的每个细节

### 🎨 场景 2: 下载并分享图表

**背景**: 用户需要将文档中的某个流程图分享给同事

**操作**:
1. 用户将鼠标移动到 Mermaid 图表上
2. 右上角显示浮动工具栏
3. 点击下载按钮（📷）
4. 浏览器自动下载 PNG 图片文件
5. (可选) 右键点击下载按钮可下载 SVG 格式

**结果**: 用户获得可用于分享和存档的图片文件
