## 功能介绍

Mermaid 图表工具栏为代码审查页面中的图表提供了全屏查看和下载功能。工具栏默认隐藏，鼠标悬停时显示，不干扰正常阅读。

🎯 **核心特性 1**: 浮动工具栏 - 鼠标悬停显示，不占屏幕空间
⚡ **核心特性 2**: 全屏查看 - 点击 ⛶ 按钮放大查看复杂图表
🔧 **核心特性 3**: 图片下载 - 点击 📷 按钮保存为 PNG，右键下载 SVG

**适用人群**：
- 代码审查人员 - 查看架构图和流程图
- 技术写作者 - 保存图表用于文档
- 项目成员 - 分享和讨论设计图表

## 快速开始

### 前置条件

- [x] 访问代码审查页面 (src/views/aicr/index.html)
- [x] 页面中有 Mermaid 代码块渲染的图表
- [x] 使用现代浏览器（Chrome/Edge/Firefox/Safari）

### 30秒上手

1. **找到图表** - 在文档中找到已渲染的 Mermaid 图表
2. **悬停显示工具栏** - 将鼠标移动到图表上，右上角出现工具栏
3. **全屏查看** - 点击 ⛶ 按钮，图表在全屏模式下显示
4. **退出全屏** - 点击右上角 ✕ 或按 ESC 键
5. **下载图片** - 点击 📷 按钮保存 PNG 图片

🎉 **完成！** 你已掌握基本使用方法。

## 操作场景

### 📋 场景 1: 全屏查看复杂架构图

**适用情况**：图表包含较多细节，在正常视图下难以看清。

**操作步骤**：
1. 将鼠标移动到图表上，等待工具栏出现
2. 点击工具栏中的 ⛶ 按钮
3. 页面变暗，图表在中央放大显示
4. 使用鼠标滚轮或触摸板可滚动查看（如需要）
5. 查看完成后，点击右上角的 ✕ 按钮，或按键盘 ESC 键退出

**预期结果**：
- 图表在半透明黑色背景上居中显示
- 图表保持清晰度，不会模糊
- 退出全屏后恢复原始视图

**注意事项**：
- ✅ 全屏模式下图表尺寸更大，便于查看细节
- ✅ 可以通过多种方式退出（按钮/ESC/点击背景）
- ❌ 全屏模式下无法与页面其他元素交互

---

### 🎨 场景 2: 下载 PNG 图片用于分享

**适用情况**：需要将图表保存到本地，用于分享或插入文档。

**操作步骤**：
1. 将鼠标移动到图表上，等待工具栏出现
2. 点击工具栏中的 📷 按钮
3. 浏览器自动下载 PNG 图片文件
4. 在下载文件夹中找到文件名格式为 `mermaid-diagram-1234567890.png` 的文件

**预期结果**：
- 图片文件成功下载
- 图片质量清晰（2倍缩放）
- 文件名包含时间戳，便于管理

**注意事项**：
- ✅ PNG 格式适合嵌入文档和分享
- ✅ 图片背景为白色，便于打印
- ❌ 大尺寸图表的 PNG 文件可能较大

---

### 🔧 场景 3: 下载 SVG 格式（右键菜单）

**适用情况**：需要矢量格式图片，用于进一步编辑或高质量打印。

**操作步骤**：
1. 将鼠标移动到图表上，等待工具栏出现
2. 在 📷 按钮上点击鼠标右键
3. 浏览器自动下载 SVG 图片文件
4. 在下载文件夹中找到 SVG 文件

**预期结果**：
- SVG 矢量图文件成功下载
- 可使用矢量编辑软件（如 Illustrator、Inkscape）编辑
- 可任意缩放不失真

**注意事项**：
- ✅ SVG 格式可无限缩放，质量最高
- ✅ 文件体积通常比 PNG 小
- ❌ 不是所有文档编辑软件都支持 SVG 插入

---

### 💡 场景 4: 查看多个图表

**适用情况**：同一页面中有多个 Mermaid 图表需要查看或下载。

**操作步骤**：
1. 在第一个图表上悬停，使用工具栏功能
2. 将鼠标移动到第二个图表上
3. 该图表的工具栏会自动显示
4. 每个图表都有独立的工具栏，互不影响

**预期结果**：
- 每个图表都有自己的工具栏
- 操作一个图表不影响其他图表

**注意事项**：
- ✅ 每个图表独立操作
- ✅ 全屏模式同时只能显示一个图表
- ❌ 不要同时尝试操作多个图表

## 常见问题

### 💡 基础问题

**Q: 工具栏为什么不显示？**
A: 工具栏默认隐藏，需要将鼠标移动到图表区域内才会显示。请确保鼠标完全悬停在图表上，而不是只在边缘。

**Q: 为什么全屏后看不到关闭按钮？**
A: 关闭按钮在右上角。如果看不到，请尝试：
- 滚动页面查看
- 按 ESC 键退出
- 点击全屏背景的任意区域

**Q: 下载的图片在哪里？**
A: 图片保存在浏览器默认下载文件夹中，通常是：
- Windows: `C:\Users\<用户名>\Downloads`
- macOS: `/Users/<用户名>/Downloads`
- Linux: `/home/<用户名>/Downloads`
文件名格式为 `mermaid-diagram-<时间戳>.png`。

---

### ⚙️ 高级问题

**Q: 下载的 PNG 图片尺寸太小，能调整吗？**
A: 当前默认使用 2 倍缩放以保证清晰度。如需不同尺寸，可以先下载 SVG 格式，然后使用矢量编辑软件调整大小。

**Q: 可以同时打开多个图表的全屏模式吗？**
A: 不可以。全屏模式同时只能显示一个图表。查看完一个后关闭，再打开另一个。

**Q: SVG 下载后用什么软件打开？**
A: SVG 是标准矢量格式，可以用以下软件打开：
- 浏览器（Chrome/Firefox/Safari/Edge）- 直接拖入
- 矢量编辑软件（Adobe Illustrator、Inkscape、Figma）
- 代码编辑器（VS Code 等）- 查看源码

---

### 🔧 故障排查

**Q: 点击下载按钮没反应？**
A: 请尝试：
1. 检查浏览器是否阻止了下载
2. 查看浏览器地址栏右侧是否有下载拦截提示
3. 尝试右键下载 SVG 格式作为替代

**Q: 全屏模式下图表不显示？**
A: 请尝试：
1. 按 ESC 退出后重试
2. 刷新页面后再次操作
3. 检查浏览器控制台是否有错误（按 F12 打开）

**Q: 下载的 PNG 图片是空白的？**
A: 这可能是 SVG 转 PNG 时的问题。请尝试：
1. 右键下载 SVG 格式作为替代
2. 在浏览器中打开 SVG 后截图
3. 如果问题持续，请报告此问题

## 技巧与提示

### 💡 实用技巧

1. **快速退出全屏** - 按 ESC 键比找关闭按钮更快
2. **两种格式按需选择** - 分享用 PNG，编辑用 SVG
3. **多图表时移动鼠标** - 将鼠标在图表间移动，工具栏会自动切换
4. **全屏下也可截图** - 全屏模式下更适合截取高质量屏幕截图

### ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| ESC | 退出全屏模式 |

> 目前仅支持 ESC 快捷键，更多快捷键可能在未来版本添加。

### 📚 最佳实践

1. **技术文档插图** - 下载 PNG 格式插入文档，兼容性最好
2. **演示幻灯片** - 下载 SVG 格式，可在 Keynote/PowerPoint 中无损缩放
3. **存档备份** - 建议同时保存 SVG 格式，方便未来编辑
4. **分享讨论** - PNG 适合直接发送，大多数设备都能打开查看
