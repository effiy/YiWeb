## 功能介绍

Mermaid 图表工具栏为 Markdown 预览中的图表提供便捷的交互能力，让您能够轻松地全屏查看复杂图表和下载图片用于分享。

### 核心特性

- 🎯 **全屏查看**: 复杂图表放大看清每个细节
- ⚡ **一键下载**: PNG 格式，直接用于文档和分享
- 🔧 **SVG 导出**: 右键下载矢量格式，可编辑
- 📖 **优雅交互**: 悬停显示，不干扰阅读

### 谁应该使用

- 代码审查人员：查看架构图、流程图
- 技术文档作者：导出图表用于文档
- 团队协作成员：分享图表给他人
- 需要阅读复杂图表的任何用户

## 快速开始

### 前置条件

- [x] 访问 `src/views/aicr/index.html` 或任何包含 MarkdownView 的页面
- [x] 页面中至少有一个 mermaid 代码块

### 30秒上手

1. **打开包含 mermaid 的文档**
   - 导航到代码审查页面
   - 打开包含 mermaid 图表的 Markdown 内容

2. **查看工具栏**
   - 将鼠标移动到图表上
   - 观察右上角出现的工具栏

3. **尝试全屏**
   - 点击「⛶」按钮
   - 在全屏模式下查看
   - 按 ESC 或点击「✕」关闭

4. **尝试下载**
   - 点击「📷」按钮
   - 检查下载的 PNG 图片

🎉 恭喜！您已掌握基本使用方法！

## 操作场景

### 📋 场景 1: 全屏查看复杂架构图

**适用情况**:
图表包含大量节点和连线，在默认尺寸下文字太小看不清楚。

**操作步骤**:
1. 将鼠标移动到图表区域
2. 等待工具栏出现（约 0.2 秒）
3. 点击工具栏左侧的「⛶」全屏按钮
4. 在全屏模式下查看图表细节
5. 完成后，按键盘 ESC 键，或点击右上角的「✕」按钮，或直接点击黑色背景区域关闭

**预期结果**:
- 图表在黑色半透明背景上居中显示
- 最大尺寸不超过屏幕的 90%
- 保持清晰锐利

**注意事项**:
- ✅ 全屏模式下可以滚动页面（如果需要）
- ❌ 不要在全屏模式下刷新页面，会退出全屏
- ✅ 三种关闭方式都可以，选您喜欢的

---

### 📋 场景 2: 下载 PNG 用于文档

**适用情况**:
需要将 mermaid 图表插入项目文档、PPT、邮件等进行分享。

**操作步骤**:
1. 将鼠标移动到图表区域
2. 等待工具栏出现
3. 点击工具栏右侧的「📷」下载按钮
4. 在浏览器下载栏中找到下载的文件
5. 插入到您的文档中

**预期结果**:
- 文件名格式: `mermaid-diagram-1234567890123.png`
- 背景为白色，适合文档使用
- 清晰度较高（2x 缩放）

**注意事项**:
- ✅ PNG 格式兼容性最好，推荐使用
- ❌ 如果图表特别复杂，下载可能需要几秒
- ✅ 下载图片的背景是白色，不是透明

---

### 📋 场景 3: 下载 SVG 用于编辑

**适用情况**:
需要对图表进行进一步的编辑、调整样式，或者需要最高质量的矢量格式。

**操作步骤**:
1. 将鼠标移动到图表区域
2. 等待工具栏出现
3. **右键点击**「📷」下载按钮（不是左键）
4. 在浏览器下载栏中找到下载的 SVG 文件
5. 使用设计工具（如 Figma、Sketch、Illustrator）打开编辑

**预期结果**:
- 文件名格式: `mermaid-diagram-1234567890123.svg`
- 矢量格式，无限放大不失真
- 可在支持 SVG 的工具中编辑

**注意事项**:
- ✅ SVG 是矢量格式，适合需要编辑的场景
- ❌ 不是所有工具都支持 SVG，确认后再使用
- ✅ SVG 文件体积通常比 PNG 小

---

### 📋 场景 4: 查看多个图表

**适用情况**:
文档中有多个 mermaid 图表，需要逐个查看或下载。

**操作步骤**:
1. 滚动到第一个图表
2. 悬停显示工具栏，进行操作
3. 滚动到第二个图表
4. 重复操作...

**预期结果**:
- 每个图表都有独立的工具栏
- 工具栏之间互不干扰
- 可以同时打开多个全屏（但建议一次一个）

**注意事项**:
- ✅ 每个图表独立管理，操作不会互相影响
- ✅ 全屏模式一次只能显示一个图表
- ❌ 不要快速连续点击多个按钮，可能错过动画

## 常见问题

### 💡 基础问题

**Q: 工具栏在哪里？我看不到。**

A: 工具栏默认是隐藏的，您需要将鼠标移动到图表区域上，它才会在右上角淡入显示。移开鼠标后会自动隐藏。

**Q: 工具栏会一直显示吗？**

A: 不会。只有当鼠标悬停在图表区域内时才会显示，移开后 0.2 秒内淡出隐藏。这样设计是为了不干扰阅读。

**Q: 我可以让工具栏一直显示吗？**

A: 目前不支持。工具栏设计为悬停显示，保持界面简洁。

**Q: 全屏模式下图表还是太小怎么办？**

A: 您可以使用浏览器的缩放功能 (Ctrl/Cmd + +) 在全屏模式下进一步放大。

---

### ⚙️ 高级问题

**Q: 下载的 PNG 背景可以是透明的吗？**

A: 目前默认是白色背景，这是为了确保在所有文档和演示软件中都能正常显示。如果需要透明背景，请使用右键下载 SVG 格式，然后在设计工具中调整。

**Q: 可以自定义下载的文件名吗？**

A: 目前文件名自动生成，格式为 `mermaid-diagram-{timestamp}.png`。您可以在下载后重命名文件。

**Q: SVG 下载后用什么软件打开？**

A: 可以用：
- 现代浏览器 (Chrome, Firefox, Safari, Edge)
- 设计工具 (Figma, Sketch, Adobe Illustrator)
- 代码编辑器 (VS Code, Sublime)
- 任何支持 SVG 的软件

**Q: 工具栏按钮的顺序可以调整吗？**

A: 目前按钮顺序是固定的：全屏在左，下载在右。

---

### 🔧 故障排查

**Q: 点击下载按钮没有反应？**

A: 请检查：
1. 浏览器是否阻止了弹出式下载
2. 尝试右键下载 SVG 作为替代
3. 打开浏览器开发者工具 (F12) 查看 Console 是否有错误

**Q: 下载的 PNG 是空白的？**

A: 可能是图表太复杂或包含外部资源。请尝试：
1. 右键下载 SVG 格式
2. 如果问题持续，请截图作为替代方案

**Q: 全屏模式打不开？**

A: 请确认：
1. 图表已完全渲染（不是加载状态）
2. 点击的是工具栏中的「⛶」按钮，不是图表本身
3. 检查浏览器 Console 是否有错误

**Q: ESC 键不能退出全屏？**

A: 尝试其他关闭方式：
1. 点击右上角的「✕」按钮
2. 点击黑色背景区域
3. 如果都不行，刷新页面

## 技巧与提示

### 💡 实用技巧

1. **快速查看多个图表**:
   - 将鼠标在图表之间缓慢移动，工具栏会平滑切换显示

2. **下载管理**:
   - 下载多个图表时，文件名包含时间戳，可以轻松区分先后

3. **全屏阅读**:
   - 全屏模式下按 ESC 是最快的退出方式

4. **文档插入**:
   - PNG 格式适合 Word、PPT、邮件
   - SVG 格式适合网页、设计文档

### ⌨️ 快捷键

| 快捷键 | 功能 | 生效范围 |
|--------|------|---------|
| ESC | 关闭全屏模式 | 全屏模式下 |

### 📚 最佳实践

1. **分享图表给团队**:
   - 推荐使用 PNG 格式，兼容性最好
   - 如果需要编辑，附带 SVG 版本

2. **文档中的图表**:
   - 先用 mermaid 代码维护源文件
   - 导出 PNG 插入文档
   - 这样更新时只需重新渲染，不用重画

3. **复杂图表处理**:
   - 先用全屏模式仔细检查
   - 确认无误后再下载
   - 特别复杂的图表建议用 SVG 格式保存

4. **命名管理**:
   - 下载后立即重命名，使用有意义的文件名
   - 例如: `系统架构图-v1.png`、`登录流程图.png`
