## §3 信任边界

| 边界 | 跨越方向 | 数据流 | 校验点 | 当前状态 |
|------|---------|--------|--------|---------|
| CSS 文件 → 浏览器渲染引擎 | 内部 | `aicrPage/index.css` → DOM 样式计算 | CSS 文件由项目源码控制，不加载外部样式表 | 已加固 |
| 浏览器扩展 → 页面样式 | 外部入站 | 扩展注入的 CSS 规则 → 页面渲染 | 浏览器扩展的 CSS 注入超出应用控制范围；使用特异性选择器 `.aicr-meta-filter-actions` 增加覆盖难度 | 已缓解 |
| 全局 CSS 变量 → 局部元素 | 内部 | `var(--yi-bg)` → 元素 `background` | 变量值由 `src/views/aicr/styles/` 中的主题定义控制 | 已加固 |
| z-index 层级 → 页面层叠 | 内部 | 数值 `3` → 渲染层叠顺序 | 项目中当前已知最大 z-index 为 2；3 留有空间 | 已加固 |
