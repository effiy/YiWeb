## §1 使用场景

### 场景 1: 视图开发者引用通用组件

**角色**: 视图开发者
**目标**: 在新视图中引入 YiModal、YiTag、YiLoading 等组件

```mermaid
flowchart LR
    A["查看 componentLoader 签名"] --> B["在 createBaseView 中声明组件"]
    B --> C["传入 componentModules 路径数组"]
    C --> D["ComponentRegistry 加载"]
    D --> E{"加载成功?"}
    E -->|"是"| F["JS/CSS/HTML 注册到 window"]
    E -->|"失败"| G["超时警告 + 降级占位"]
    F --> H["模板中通过标签名引用"]
```

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 在 components 数组声明 `'YiModal', 'YiButton'` | 组件名称注册 |
| 2 | 在 componentModules 数组写 CDN 路径 | 组件文件加载 |
| 3 | 在 template.html 中写 `<yi-modal ...>` | 组件渲染 |
| 4 | 查看组件 props | props 绑定生效 |

---

### 场景 2: 创建新视图

**角色**: 视图创建者
**目标**: 使用 createBaseView 创建符合规范的新视图

```mermaid
flowchart LR
    A["创建 index.js"] --> B["定义 store"]
    B --> C["定义 useComputed"]
    C --> D["定义 useMethods"]
    D --> E["声明 components + componentModules"]
    E --> F["调用 createBaseView"]
    F --> G["传入 data/onMounted/onUnmounted"]
    G --> H["视图挂载到指定元素"]
```

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | import createBaseView | 模块正常加载 |
| 2 | 定义 store = createStore() | vueRef 响应式数据 |
| 3 | 实现 useComputed(store) | computed 属性就绪 |
| 4 | 实现 useMethods(store) | methods 方法就绪 |
| 5 | createBaseView(...) | 视图初始化完成 |

### 场景 3: 渲染 Markdown 文档

**角色**: 组件开发者
**目标**: 使用 Markdown 引擎渲染内容，自动处理 Mermaid 图表和 XSS 过滤

```mermaid
flowchart LR
    A["接收 Markdown 文本"] --> B["调用 renderMarkdownHtml"]
    B --> C["marked 解析"]
    C --> D["插件管道"]
    D --> E["SanitizePlugin 过滤"]
    D --> F["Container/Accordion 容器"]
    D --> G["TOC 生成"]
    D --> H["Mermaid 图表嵌入"]
    H --> I["v-html 渲染结果"]
```
