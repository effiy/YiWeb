# 故事任务

> **基线类型**：问题空间 · **版本**：1.0.0 · **生成日期**：2026-05-26

## 概述

CDN 模块是 YiWeb 的通用基础设施层，提供共享 UI 组件、图标系统、Markdown/Mermaid 渲染引擎、CSS 样式体系以及工具库。所有视图（aicr / claude / story）均依赖此层进行组件注册和功能复用，实现视图间的零冗余共享。

### 主要价值

- 🧩 **组件库** — 14 个通用 UI 组件（按钮/标签/模态框/表单/反馈），覆盖所有视图的 UI 需求
- 🎨 **样式体系** — 全局 CSS Reset + 主题 + 工具类 + 组件样式
- 📝 **Markdown 渲染** — 插件化 Markdown 引擎（Sanitize / Mermaid / TOC / Container / Accordion）
- 📊 **Mermaid 图表** — 图表渲染 + 工具栏（缩放/全屏/下载/复制/错误修复）
- 🔧 **工具库** — 40+ 工具文件（DOM/事件/HTTP/存储/时间/验证/动画/性能）
- 🏗️ **视图框架** — createBaseView + ComponentRegistry 组件加载系统

| FP1 | 组件注册系统 — ComponentRegistry + componentLoader，支持按路径加载和注册到 window | P0 |
| FP2 | 基础视图 — createBaseView 提供 store + computed + methods + 组件注册 + 生命周期 | P0 |
| FP3 | 按钮组件 — YiButton 6 变体 + YiIconButton 2 变体 | P0 |
| FP4 | 标签组件 — YiTag 6 颜色 + 可关闭 + 可点击 + 激活态 | P0 |

| FP10 | 图标系统 — YiIcon 语义名称映射 Font Awesome 类名 | P0 |

| FP12 | 工具库 — 日志/错误/HTTP/存储/事件/验证/时间/动画/性能 | P0 |
| FP13 | 搜索栏 — SearchHeader 搜索输入 + 历史记录 + 键盘导航 | P1 |

### 业务规则

| R# | 描述 |
|----|------|
| R1 | 所有组件通过 componentLoader 加载，注册到 `window.__components__` |
| R2 | Markdown 渲染强制启用 SanitizePlugin 防 XSS |
| R3 | 工具库禁止裸 console.log，统一使用 logInfo/logWarn/logError |
| R4 | 组件 CSS 作用域通过 BEM 类名约定隔离 |

## §4 AC

| AC# | Given | When | Then |
|-----|-------|------|------|
| AC1 | 任一视图启动 | createBaseView 初始化 | 组件按路径加载并注册 |
| AC2 | 需要模态框 | 调用 YiModal.show() | 模态框展示，Esc/遮罩可关闭 |
| AC3 | Markdown 内容含 XSS | 渲染内容 | SanitizePlugin 过滤危险标签 |
| AC4 | Markdown 含 Mermaid 代码块 | 渲染内容 | Mermaid 图表渲染并带工具栏 |
| AC5 | 空列表数据 | 组件渲染 | YiEmptyState 显示提示文字 |
| AC6 | API 错误 | 组件渲染 | YiErrorState 显示错误 + 重试按钮 |
| AC7 | 需要异步操作反馈 | 触发 loading 状态 | YiLoading 展示动画和文字 |
