## §3 关键决策

| # | 决策 | 理由 |
|---|------|------|
| 1 | L0 展示层独立为一层 | HTML 模板不含 JS 逻辑，与 L1 视图层职责明确不同 |
| 2 | 依赖方向严格单向 | 上层依赖下层，防止基础设施层被业务逻辑污染 |
| 3 | 分层判定以目录位置为主 | 项目目录结构直接反映分层意图，减少主观判定 |
