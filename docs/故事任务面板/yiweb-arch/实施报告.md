## §7 可操作验证

### 验证步骤

1. 进入项目根目录
2. 验证文档存在性：`ls docs/故事任务面板/yiweb-arch/`
3. 验证版头格式：`head -5 docs/故事任务面板/yiweb-arch/*.md | grep -E "^> v[0-9]+\.[0-9]+\.[0-9]+"`
4. 验证导航链接：`grep "导航" docs/故事任务面板/yiweb-arch/*.md`
5. 验证模块存在性：逐一检查 §2 中各模块的验证命令
6. 验证安全合规：执行 §2 中 L2 服务层的 P0 检查命令

### 快速自检

```bash
# 文档基线完整性
ls docs/故事任务面板/yiweb-arch/ | wc -l  # 预期 ≥5

# 版头格式一致性
grep -l "^> v" docs/故事任务面板/yiweb-arch/*.md | wc -l  # 预期 5

# 导航链完整性
grep -l "导航" docs/故事任务面板/yiweb-arch/*.md | wc -l  # 预期 5

# 主要价值
grep -c "^\- " docs/故事任务面板/yiweb-arch/*.md  # 每文件 ≥4
```
