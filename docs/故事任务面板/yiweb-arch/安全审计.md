## 资产识别

| 资产 | 存储位置 | 敏感度 | 暴露面 |
|------|---------|:--:|------|
| X-Token | localStorage | 高 | 浏览器 DevTools、XSS |
| 会话数据 | 内存（store state） | 中 | 浏览器内存转储 |
| 用户输入（聊天/搜索） | 内存 + 网络传输 | 中 | 网络抓包、XSS |
| 项目源码路径 | 文档 + 内存 | 低 | 公开仓库 |
| AI 对话内容 | 内存 + API 传输 | 中 | 网络抓包 |
| 环境配置 | config.js | 低 | 公开仓库 |

## STRIDE 威胁建模

### Spoofing（身份伪造）

| 威胁 | 影响资产 | 风险 | 描述 |
|------|---------|:--:|------|
| Token 窃取 | X-Token | H | localStorage 中的 Token 可被 XSS 或恶意浏览器扩展读取 |
| 无会话管理 | X-Token | M | 无 Token 过期机制，长期有效增加泄露风险 |

**缓解措施**：
- Markdown 渲染启用 SanitizePlugin 防 XSS
- `credentials: 'omit'` 防止 Cookie 携带
- 建议：添加 Token 过期时间和刷新机制

**验证**: `grep -r "SanitizePlugin" cdn/markdown/` + `grep -r "credentials.*omit" src/core/services/`

---

### Tampering（数据篡改）

| 威胁 | 影响资产 | 风险 | 描述 |
|------|---------|:--:|------|
| CDN 劫持 | 全部前端代码 | H | ESM import 从 CDN 加载，无 SRI 完整性校验 |
| 中间人攻击 | 用户输入/API 数据 | M | HTTP 请求未强制 HTTPS（依赖服务器配置） |
| 缓存投毒 | API 响应 | M | 内存缓存无版本标记，可能返回过期数据 |

**缓解措施**：
- 生产环境强制 HTTPS
- 建议：为关键 CDN 资源添加 SRI hash

**验证**: 检查 config.js 中 prod 环境 URL 均为 HTTPS

---

### Repudiation（不可否认性）

| 威胁 | 影响资产 | 风险 | 描述 |
|------|---------|:--:|------|
| 无操作审计 | 会话数据 | L | 前端不记录用户操作日志，无法追溯 |
| 无变更日志 | 项目源码 | L | 依赖 git commit，无应用层审计 |

**缓解措施**：
- Git 提交历史提供源码级追溯
- 后端 API 应记录操作日志（前端不负责）

**验证**: `git log --oneline -5`

---

### Information Disclosure（信息泄露）

| 威胁 | 影响资产 | 风险 | 描述 |
|------|---------|:--:|------|
| 错误消息泄露 | API 响应 | M | 后端错误详情可能暴露内部路径或栈追踪 |
| Console 日志泄露 | 调试信息 | L | `logInfo/logWarn/logError` 在生产环境可能输出敏感信息 |
| 源码暴露 | 项目结构 | L | 前端代码天然可读，不包含密钥即可 |

**缓解措施**：
- 生产环境禁用详细错误输出
- 日志工具 `cdn/utils/core/log.js` 应区分环境

**验证**: `grep -r "console\\.log" src/` — 检查是否有裸 console.log

---

### Denial of Service（拒绝服务）

| 威胁 | 影响资产 | 风险 | 描述 |
|------|---------|:--:|------|
| 客户端资源耗尽 | 浏览器 | M | 无请求频率限制，大量并发可能耗尽浏览器资源 |
| 内存泄漏 | 内存 | M | vueRef 响应式依赖未正确清理可能导致内存泄漏 |
| 超大响应 | 内存/渲染 | L | 无响应大小限制，超大 JSON 可能导致渲染卡死 |

**缓解措施**：
- `AbortController` 支持请求取消
- `onBeforeUnmount` 生命周期清理定时器和事件监听
- 建议：添加请求去重和响应大小限制

**验证**: `grep -r "AbortController" src/` + `grep -r "beforeUnmount\|onBeforeUnmount" src/views/`

---

### Elevation of Privilege（权限提升）

| 威胁 | 影响资产 | 风险 | 描述 |
|------|---------|:--:|------|
| 401 绕过 | X-Token | M | 若 authErrorHandler 有 bug，可能跳过认证 |
| Token 伪造 | X-Token | L | 前端无法伪造有效 Token（依赖后端验证） |
| 跨视图数据访问 | 会话数据 | L | 视图间隔离，但共享 store 可能意外暴露数据 |

**缓解措施**：
- 401 处理有 2s 冷却防重复弹窗
- 视图间隔离（独立目录，不交叉引用）
- 后端验证 Token 有效性（前端只负责传递）

**验证**: `grep -r "handle401Error\|clearToken\|openAuth" src/core/services/`

## 信任边界

```
┌─────────────────────────────────────────────────────┐
│                    浏览器沙箱                          │
│  ┌──────────┐  ┌──────────┐  ┌──────────────────┐  │
│  │ 视图层    │  │ 服务层    │  │ localStorage     │  │
│  │ (可信)   │  │ (可信)   │  │ (半可信)         │  │
│  └────┬─────┘  └────┬─────┘  └────────┬─────────┘  │
│       │              │                 │             │
│  ─ ─ ─│─ ─ ─ ─ ─ ─ ─│─ ─ ─ ─ ─ ─ ─ ─ │─ ─ ─ ─ ─  │
│       │              │                 │             │
│  ┌────┴──────────────┴─────────────────┴─────────┐  │
│  │              网络边界（不可信）                  │  │
│  │  CDN 资源 · API 响应 · Ollama 流式             │  │
│  └────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────┘
```

| 边界 | 方向 | 风险等级 | 缓解 |
|------|------|:--:|------|
| 视图/服务 → CDN | 出站 | M | SRI（建议）、HTTPS |
| 视图/服务 → API | 出站 | M | Token 认证 + HTTPS |
| API 响应 → 视图 | 入站 | M | SanitizePlugin 防 XSS |
| localStorage → 应用 | 读入 | H | Token 不设过期（风险） |
| 应用 → localStorage | 写出 | M | 仅存储 Token/env/debug |

## 合规检查

| # | 检查项 | 状态 | 证据 |
|---|--------|:--:|------|
| 1 | 输入校验 — 所有用户输入点有校验或转义 | ✅ | `SanitizePlugin` + `encodeURIComponent` |
| 2 | 认证 — 所有 API 请求携带 X-Token | ✅ | `getAuthHeaders()` 统一添加 |
| 3 | 数据保护 — 敏感数据不入 localStorage（除 Token） | ✅ | 会话数据仅存内存 |
| 4 | 日志 — 使用统一日志工具而非裸 console | ✅ | `logInfo/logWarn/logError` |
| 5 | 错误处理 — 不向用户暴露内部错误详情 | ⚠️ | 依赖后端返回格式 |
| 6 | 第三方 — CDN 资源有降级策略 | ✅ | ComponentRegistry 超时降级 |
