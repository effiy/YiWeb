## §1 功能用例

| # | 用例 | 步骤 | 预期结果 |
|---|------|------|---------|
| TC1 | 默认显示二级标签 | 打开 AICR 页面，观察 Sidebar | Sidebar 标签区显示所有二级目录名（如 components, utils, hooks） |
| TC2 | 二级标签筛选文件树 | 点击 Sidebar 的「components」标签 | 文件树只显示含 components 子目录的一级目录 |
| TC3 | 清除后恢复无筛选 | 点击「清除筛选」或再次点击已选标签 | 文件树恢复无筛选状态 |
| TC4 | 展开/折叠标签 | 点击「展开」按钮 | 显示所有标签；点击「收起」恢复默认 8 个 |
| TC5 | 吸顶效果 | 滚动代码区域 | Sidebar 保持在视口顶部，不随页面滚动消失 |
| TC6 | 空状态 | 文件树无二级目录 | Sidebar 标签区隐藏 |

## §2 边界用例

| # | 用例 | 预期结果 |
|---|------|---------|
| TC7 | 文件树为空 | 标签区不显示 |
| TC8 | 只有文件没有文件夹 | 标签区不显示 |
| TC9 | 窄屏模式（≤950px） | Sidebar 变为顶部横条，不吸顶 |
