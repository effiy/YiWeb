## §1 正常路径

### TC-N01: 项目列表加载

| Given | 远端 API 返回项目列表 | When | 页面加载 | Then | 项目卡片列表渲染 |

### TC-N02: 点击项目展示详情

| Given | 列表已加载 | When | 点击项目卡片 | Then | 详情卡片展示 |

### TC-N03: 返回列表

| Given | 详情卡片展示中 | When | 点击返回 | Then | 详情关闭，列表显示 |

---

## §2 边界/异常

### TC-E01: 远端 API 不可达

| Given | 网络断开 | When | 页面加载 | Then | 错误状态 + 重试按钮 |

### TC-E02: 详情加载失败

| Given | 列表正常 | When | 点击项目后详情 API 失败 | Then | 错误占位，不影响列表 |
