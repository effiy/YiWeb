## §1 使用场景

### 场景 1: 项目概览浏览

**角色**: Claude 使用者
**目标**: 查看所有 Claude 项目及其配置完备度

```mermaid
flowchart LR
    A["加载面板"] --> B{"API 响应?"}
    B -->|"成功"| C["卡片网格渲染"]
    B -->|"失败"| D["错误状态 + 重试"]
    C --> E["浏览项目卡片"]
    E --> F["查看标签"]
    F --> G["了解各项目配置状态"]
```

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 打开 Claude 面板 | 加载动画 → 卡片网格展示 |
| 2 | 浏览卡片 | 每个卡片展示项目名和配置状态标签 |
| 3 | 查看空状态 | 无项目时显示空状态引导 |

---

### 场景 2: 查看项目详情

**角色**: Claude 使用者
**目标**: 深入了解特定项目的配置详情

```mermaid
flowchart LR
    A["项目列表已加载"] --> B["点击项目卡片"]
    B --> C{"详情加载?"}
    C -->|"成功"| D["详情卡片展示"]
    C -->|"失败"| E["错误占位"]
    D --> F["查看 CLAUDE.md 等配置状态"]
    F --> G["点击返回"]
    G --> H["回到列表"]
```

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| 1 | 点击项目卡片 | 选中高亮，详情卡片弹出/展开 |
| 2 | 查看详情 | 展示 CLAUDE.md、settings、skills、agents、memory 状态 |
| 3 | 点击返回 | 详情关闭，列表恢复 |

---

### 场景 3: 筛选与排序查找

**角色**: Claude 管理者
**目标**: 按配置完备度筛选和排序，定位需要关注的项目

```mermaid
flowchart LR
    A["项目列表已加载"] --> B["输入搜索关键词"]
    B --> C["实时过滤卡片"]
    C --> D["选择配置状态筛选"]
    D --> E["叠加过滤"]
    E --> F["切换排序方式"]
    F --> G["结果重新排列"]
```
