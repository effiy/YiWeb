## §1 使用场景

### 场景 1: 查看 Claude 项目概览

**角色**: Claude 使用者
**目标**: 打开面板，浏览所有项目

```mermaid
flowchart LR
    A["打开面板"] --> B["加载项目列表"]
    B --> C{"数据?"}
    C -->|"有数据"| D["卡片列表展示"]
    C -->|"空"| E["空状态提示"]
    C -->|"错误"| F["错误+重试"]
```

### 场景 2: 查看项目详情

**角色**: Claude 使用者
**目标**: 点击项目查看详细信息

```mermaid
flowchart LR
    A["浏览项目列表"] --> B["点击项目卡片"]
    B --> C["加载详情"]
    C --> D["详情卡片展示"]
    D --> E["查看后返回"]
    E --> A
```
