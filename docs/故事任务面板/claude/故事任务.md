# 故事任务

> **基线类型**：问题空间 · **版本**：1.0.0 · **生成日期**：2026-05-26

## 概述

Claude 面板是 YiWeb 的 Claude 项目管理视图。用户可查看所有 Claude 项目的状态概览，通过卡片网格浏览项目信息，点击项目查看详细信息。面板与 story 面板共享相同的 `createBaseView` 架构模式，数据来自远端 API。

### 主要价值

- 🎯 **项目概览** — 统一管理入口，查看所有 Claude 项目配置状态
- 🔒 **只读面板** — 仅查看项目信息，不修改数据
- ⚡ **卡片展示** — 以卡片网格布局清晰呈现项目关键信息
- 📊 **详情卡片** — 点击项目展示完整的配置和状态信息

## §2 Requirements

### 功能点

| FP# | 描述 | 输入 | 输出 | 错误行为 | 优先级 |
|-----|------|------|------|---------|--------|
| FP1 | 项目列表查询 — 从 API 获取项目列表 | API 请求 | 项目数据数组 | API 失败显示错误状态 | P0 |
| FP2 | 卡片网格渲染 — 卡片形式展示项目 | 项目数据 | 卡片网格 DOM | 无数据显示空状态 | P0 |
| FP3 | 项目详情展示 — 选中项目的详细信息 | 项目 ID | 详情卡片 | 加载失败占位提示 | P0 |
| FP4 | 返回导航 — 从详情回到列表 | 点击返回 | 列表视图恢复 | — | P0 |
| FP5 | 项目名称搜索 — 实时过滤 | 搜索关键词 | 过滤后卡片 | 空搜索显示全部 | P1 |
| FP6 | 配置状态筛选 — 按 CLAUDE.md/settings/skills/agents/memory 完备度 | 筛选条件 | 过滤后项目列表 | 无筛选显示全部 | P1 |
| FP7 | 多字段排序 — 按名称/时间/文件数排序 | 排序字段 | 排序后列表 | 默认时间降序 | P1 |

### 业务规则

| R# | 描述 |
|----|------|
| R1 | 数据源为远端 API，不读本地文件系统 |
| R2 | 面板仅展示，不修改项目数据 |
| R3 | 项目卡片展示 CLAUDE.md、settings、skills、agents、memory 五种配置状态 |

### 数据约束

| 约束 | 类型 | 范围/格式 | 来源 |
|------|------|----------|------|
| 项目列表 | array | API 返回的项目数组 | 后端 API |
| 搜索关键词 | string | 任意文本，trim 匹配 | 用户输入 |
| 排序字段 | enum | name / updatedAt / fileCount / skillCount | 数据模型 |

| SC# | 描述 | 度量方式 | 目标值 | 优先级 |

## §4 范围边界

### 范围内
- 项目列表展示（卡片网格）
- 项目详情卡片
- 项目搜索与配置状态筛选
- 多字段排序

### 范围外
- 项目创建/编辑/删除（由其他工具管理）
- Claude CLI 配置编辑（设置文件由用户手动修改）
- 项目健康监控（不涉及运行时巡检）
