## 技术决策

1. **双命名空间策略**：新增 `--yi-*` 作为规范令牌，保留旧变量名映射到 `--yi-*`，确保现有组件零改动即可继承新设计。
2. **Soft UI 阴影**：将原有重阴影（rgba(0,0,0,0.4+)）降低为更柔和的层级，符合 UI UX Pro Max 对 SaaS 产品的推荐。
3. **字体升级**：引入 Inter（UI）+ JetBrains Mono（代码），替代系统默认字体栈，提升代码审查场景的可读性。
4. **无障碍预留**：`prefers-reduced-motion`、`prefers-contrast: high` 已支持；暗色模式令牌已定义。
