# 03-前端技术评审 — YiWeb-filetree-sidebar-ux

## 设计方向

Soft UI + Minimalism，暗色主题适配。

## 文件改造规格

### fileTreeLayout.css

| 项 | 改造前 | 改造后 |
|----|--------|--------|
| 工具栏按钮背景 | `linear-gradient(135deg, #6366f1, #4f46e5)` | `var(--yi-primary)` 纯色 |
| 工具栏按钮 Hover | `linear-gradient(...)` + 发光阴影 | `var(--yi-primary-hover)` + `--yi-shadow-sm` |

## 无障碍要求

- 按钮/标签最小触摸目标 44×44px
- 焦点环可见：`--yi-shadow-focus`
- `prefers-reduced-motion` 禁用所有动画
