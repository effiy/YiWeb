## 文件改造详情

### fileTreeLayout.css

- **文件**: `src/views/aicr/components/fileTree/fileTreeLayout.css`
- **改动**: 工具栏按钮去渐变 + 搜索聚焦环标准化 + 收起按钮简化
- **要点**:
  - 工具栏按钮背景: `linear-gradient(135deg, ...)` → `var(--yi-primary)` 纯色
  - Hover: 渐变+发光 → `var(--yi-primary-hover)` + `--yi-shadow-sm`
  - 危险按钮: 红色渐变 → `var(--yi-danger)` 纯色
  - 搜索框聚焦: `rgba(79,70,229,0.15)` 环 → `--yi-shadow-focus`
  - 文件树切换按钮: 移除 `::before` radial-gradient 光晕，阴影降为 `--yi-shadow-sm/md`
  - 空状态图标背景: 渐变发光 → `var(--yi-primary-subtle)`
  - `var(--bg-glass)` → `var(--yi-surface-hover)`

### fileTreeTreeBase.css

- **文件**: `src/views/aicr/components/fileTree/fileTreeTreeBase.css`
- **改动**: 移除脉冲动画 + 简化 hover + 令牌化
- **要点**:
  - `.file-tree-item:hover`: 移除 `transform: translateX(2px)` 和 `box-shadow`
  - 移除 `.highlight-file` 的 `animation: highlightFilePulse`
  - 移除 `.file-tree-highlight` 的 `animation: fileTreeHighlightPulse`
  - 移除 `.file-warning-badge` 的 `animation: warningPulse`
  - 保留关键帧定义（向后兼容），但不再默认触发
  - `var(--bg-glass)` → `var(--yi-surface-hover)` (file-size)
  - 新增 `@media (prefers-reduced-motion: reduce)`

### fileTreeTreeExtras.css

- **文件**: `src/views/aicr/components/fileTree/fileTreeTreeExtras.css`
- **改动**: 空状态简化 + 拖拽样式简化 + 标签令牌化
- **要点**:
  - 空状态卡片: 移除 `box-shadow` 和 `opacity: 0.85`，背景改为 `var(--yi-surface)` + `var(--yi-border)`
  - 空状态图标: `var(--primary)` → `var(--yi-primary)`，移除 hover 变色
  - 拖拽高亮: 移除 `transform: scale(1.02)` 和发光 `box-shadow`，改为 `rgba(var(--yi-info-rgb), 0.2)` + 虚线边框
  - 拖拽指示线: 移除 `box-shadow` 发光
  - `.file-tag:hover`: 硬编码 `rgba(255,255,255,0.1)` → `var(--yi-surface-hover)`，移除 `translateY(-1px)`
  - `.file-tag.batch-selected`: 硬编码 `#10b981` → `var(--yi-success)`
  - `var(--bg-glass)` → `var(--yi-surface-hover)` (folder-count, file-actions, file-tag)
  - 新增 `@media (prefers-reduced-motion: reduce)`

### aicrHeader/index.css

- **文件**: `src/views/aicr/components/aicrHeader/index.css`
- **改动**: 无需修改，已使用 `--yi-duration-fast`，无硬编码色值

## 令牌覆盖

- 硬编码 indigo / cyan / emerald 全面替换为 `--yi-primary-rgb` / `--yi-info-rgb` / `--yi-success-rgb`
- 所有 `transition` 统一为 `--yi-duration-*` + `--yi-easing-default`
- 阴影统一为 `--yi-shadow-*`
- 玻璃态背景 `var(--bg-glass)` 统一替换为 `var(--yi-surface-hover)`

## 可访问性

- `prefers-reduced-motion` 已覆盖 fileTree / sessionListTags 所有动画
- 焦点环使用 `--yi-shadow-focus`
- 触摸目标保持 44px（移动端 media query 已定义）
