# 04-测试用例评审 — YiWeb-filetree-sidebar-ux

## 测试策略

人工验证 + DevTools 检查清单。

## 测试用例

### TC-01: 工具栏按钮样式

| 项 | 内容 |
|----|------|
| 目的 | 确认工具栏按钮使用纯色而非渐变 |
| 步骤 | 1. 打开文件树视图<br>2. 观察新建/删除/上传/下载按钮 |
| 预期 | 纯色背景，无渐变残留 |

### TC-02: 文件树选中状态

| 项 | 内容 |
|----|------|
| 目的 | 选中/高亮使用 `--yi-*` 令牌 |
| 步骤 | 1. 点击文件树项<br>2. DevTools Computed 检查背景色 |
| 预期 | `--yi-primary-subtle` / `--yi-success-subtle` |

### TC-03: 脉冲动画移除

| 项 | 内容 |
|----|------|
| 目的 | `highlightFilePulse` / `warningPulse` 不再默认触发或已简化 |
| 步骤 | 1. 触发文件高亮/大文件警告<br>2. 观察是否有脉冲效果 |
| 预期 | 无脉冲（或仅在非 reduced-motion 下有一次性动画） |

### TC-04: 焦点状态

| 项 | 内容 |
|----|------|
| 目的 | 文件项焦点环使用 `--yi-shadow-focus` |
| 步骤 | 1. Tab 遍历文件树 |
| 预期 | 可见蓝色焦点环 |

### TC-05: 拖拽样式

| 项 | 内容 |
|----|------|
| 目的 | 拖拽高亮使用语义令牌 |
| 步骤 | 1. 拖拽文件树项 |
| 预期 | `--yi-info-subtle` 背景，无发光阴影 |

### TC-06: 会话标签

| 项 | 内容 |
|----|------|
| 目的 | 会话列表标签使用 `--yi-*` |
| 步骤 | 1. 检查标签 hover/active 态 |
| 预期 | `--yi-surface-hover` / `--yi-primary-subtle` |

## 门禁标准

- TC-01 至 TC-04 为 P0，必须全部通过
- TC-05、TC-06 为 P1，允许一次修复
