# 03-前端技术评审 — YiWeb-aicr-layout-ux

## 设计方向

基于 UI UX Pro Max 对 SaaS / AI Platform 的推荐：
- **Style**: Minimalism & Swiss Style + Soft UI Evolution
- **Effects**: Soft shadows, subtle depth, smooth transitions (150–300ms)
- **Avoid**: Heavy gradients, neon glows, decorative-only animations, pulse effects

## 文件改造规格

### layout.css

| 项 | 改造前 | 改造后 |
|----|--------|--------|
| 侧边栏阴影 | `var(--shadow-sm/md)`（旧重阴影） | `--yi-shadow-sm/md` |
| 展开把手背景 | `linear-gradient(135deg, var(--primary)..., var(--primary-dark))` | `--yi-primary` 纯色 |
| 展开把手阴影 | `0 6px 18px rgba(79,70,229,0.35)` + pulse | `--yi-shadow-md`，移除 pulse |
| 文件树切换按钮 | 紫 indigo 发光 + radial-gradient 光晕 | `--yi-surface` + `--yi-border` |
| 代码区边缘引导 | `linear-gradient(90deg, rgba(79,70,229,0.18)...)` | 移除或改用 `--yi-border` |
| 过渡 | `0.3s cubic-bezier(0.4, 0, 0.2, 1)` | `var(--yi-duration-normal)` + `var(--yi-easing-default)` |
| 搜索清除按钮 | 硬编码 `rgba(255,255,255,0.1)` | `--yi-surface-hover` |
| 滚动条 | `var(--border-primary)` | `--yi-border` |

## 无障碍要求

- 按钮/把手最小触摸目标 44×44px
- 焦点环可见：`--yi-shadow-focus`
- `prefers-reduced-motion` 禁用所有动画
- 颜色不单独传达信息
