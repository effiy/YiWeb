## 文件改造详情

### layout.css

- **文件**: `src/views/aicr/styles/layout.css`
- **改动**: 批量替换硬编码色值 + 标准化过渡 + 简化视觉效果
- **要点**:
  - 所有 `rgba(79,70,229,...)` indigo 硬编码 → `rgba(var(--yi-primary-rgb), ...)`
  - 展开把手背景: `linear-gradient(...)` → `var(--yi-primary)` 纯色
  - 移除 `handlePulse` 脉冲动画关键帧与引用
  - 代码区边缘引导: 重 indigo 渐变 →  subtle `var(--yi-border)` 渐变
  - 所有 `transition` 统一为 `var(--yi-duration-*)` + `var(--yi-easing-default)`
  - 搜索清除按钮: 硬编码 `rgba(255,255,255,0.1)` → `var(--yi-surface-hover)`
  - `prefers-reduced-motion` 扩展覆盖 chat-expand-handle、file-tree-toggle 伪元素

### index.css

- **文件**: `src/views/aicr/styles/index.css`
- **改动**: 替换硬编码 indigo + 标准化过渡
- **要点**:
  - `rgba(79,70,229,...)` → `rgba(var(--yi-primary-rgb), ...)`
  - 过渡统一为 `--yi-duration-*` / `--yi-easing-default`
  - 阴影变量 `var(--shadow-sm/md)` → `var(--yi-shadow-sm/md)`

### codePage.css

- **文件**: `src/views/aicr/styles/codePage.css`
- **改动**: 批量替换硬编码色值 + 简化空状态卡片 + 修复预存语法错误
- **要点**:
  - 全面替换硬编码 rgba 为 token-based: `--yi-primary-rgb`, `--yi-info-rgb`, `--yi-danger-rgb`, `--yi-success-rgb`, `--text-muted-rgb`, `--text-secondary-rgb`, `--bg-primary-rgb`, `--bg-secondary-rgb`, `--bg-tertiary-rgb`
  - 用户聊天气泡: `linear-gradient(...)` → `var(--yi-primary)` 纯色
  - 空状态卡片 (`pet-chat-empty-card`, `code-view-container .empty-card`): 多重渐变+玻璃态+发光线条 → `var(--yi-surface)` + `var(--yi-border)` + `var(--yi-shadow-lg)`
  - 空状态图标: 渐变发光背景 → `var(--yi-primary-subtle)` + `var(--yi-primary)` 文字
  - 修复预存语法错误: 移除 `.pet-chat-right-panel .chat-message-input:focus` 后的重复声明块
  - 过渡全面标准化

### codePage.contextModals.css

- **文件**: `src/views/aicr/styles/codePage.contextModals.css`
- **改动**: 批量替换硬编码色值 + 新增 reduced-motion
- **要点**:
  - 全面替换硬编码 rgba 为 token-based（同 codePage.css 映射）
  - 遮罩 `rgba(0,0,0,0.55)` → `rgba(var(--bg-primary-rgb), 0.55)`
  - 模态框背景/边框/文字全面 token 化
  - 新增 `@media (prefers-reduced-motion: reduce)` 覆盖所有交互元素

### welcomeCard.css

- **文件**: `src/views/aicr/styles/welcomeCard.css`
- **改动**: 全量重写
- **要点**:
  - 移除 cyan/blue 渐变背景
  - 使用 `var(--yi-surface)` 背景 + `var(--yi-info)` 左边框
  - 标签: `rgba(59,130,246,0.15)` → `var(--yi-primary-subtle)`
  - 全部 spacing/font/shadow 使用 `--yi-*` 令牌
  - 新增 `prefers-reduced-motion`

## 令牌覆盖

暗色 aicr UI 采用混合策略：
- 保留 legacy 变量（`--bg-secondary`, `--text-primary`, `--border-primary` 等）因其已正确映射到暗色值
- 硬编码 rgba 全面替换为 token-based RGB 变量
- 阴影使用 `--yi-shadow-*`（在暗色下需后续增强）

## 可访问性

- `prefers-reduced-motion` 已覆盖所有动画（含脉冲、滑入、hover transform）
- 焦点环保留可见性
- 触摸目标在 layout.css 中已定义 44px 规则
