# 04-测试用例评审 — YiWeb-search-history-suggestions

## 测试策略

- **单元测试**：SearchHandler 纯逻辑（历史增删、过滤、导航）
- **组件测试**：SearchHeader 面板渲染、事件交互、Vue 响应式
- **无障碍测试**：axe-core 静态扫描 + 键盘流人工验证
- **视觉回归**：面板展开/收起、暗色模式截图对比

## 测试用例
