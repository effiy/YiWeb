## 文件变更详情

### cdn/styles/theme.css

- 新增 7 个搜索面板 CSS 变量：
  - `--search-panel-bg`
  - `--search-panel-border`
  - `--search-panel-shadow`
  - `--search-panel-item-hover`
  - `--search-panel-item-active`
  - `--search-panel-max-height`

### cdn/utils/browser/events.js

- `SearchHandler` 类增强：
  - 新增状态：`activeIndex`、`onActiveIndexChange`、`onShowPanel`、`onHidePanel`
  - 补全 `handleArrowKeys`：支持键盘循环导航
  - 更新 `handleFocus` / `handleBlur`：通过回调通知外部 UI
  - 新增 `removeHistoryItem(query)`：单条删除
  - 新增 `clearHistory()`：清空全部
  - 新增 `filterHistory(query)`：客户端过滤
- 移除 3 处 TODO 标记

### cdn/components/business/SearchHeader/index.css

- 新增 `.search-panel` 及其子元素完整样式：
  - 面板定位、动画、暗色/毛玻璃适配
  - 历史项 hover / active 状态
  - 删除按钮仅 hover 显示
  - 空态提示、底部「清空全部」
  - `prefers-reduced-motion` 降级

### cdn/components/business/SearchHeader/template.html

- 为 `textarea` 新增 `@focus`、`@blur`、`:aria-expanded`、`aria-controls`、`aria-activedescendant`
- 新增 `.search-panel` 结构：
  - `role="listbox"` 容器
  - 历史项列表 `role="option"`、`:aria-selected`
  - 匹配高亮文本（`v-html` 经 `escapeHtml` 净化）
  - 单项删除按钮、空态、清空全部

### cdn/components/business/SearchHeader/index.js

- 导入 `SearchHandler`
- 新增响应式状态：`isPanelVisible`、`activeIndex`、`filteredHistory`、`displayItems`
- `Vue.onMounted` 初始化 `SearchHandler`（数据层，Vue 接管事件）
- `handleInput`：输入时显示面板并重置高亮
- `handleFocus`：聚焦时若历史非空则显示面板
- `handleBlur`：延迟 200ms 关闭面板
- `handleKeydown`：集成 ArrowUp/ArrowDown 循环导航、Enter 选中/搜索、Escape 关闭
- `selectItem` / `deleteItem` / `clearAllHistory`：面板交互
- `highlightMatch`：安全高亮（`escapeHtml` + 正则匹配）
- `clearSearch`：重置面板状态

## 性能考量

- `filteredHistory` 为 `Vue.computed`，仅在 `searchQuery` 或 `searchHistory` 变化时重算
- `displayItems` 直接代理 `filteredHistory`，无额外计算
- 历史上限 10 条，过滤操作复杂度 O(n)，可忽略
- 面板动画使用 CSS transform/opacity，GPU 合成层友好
