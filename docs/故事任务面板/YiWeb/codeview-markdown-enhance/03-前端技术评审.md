# 03-前端技术评审 — YiWeb-codeview-markdown-enhance

## 设计方向

Soft UI + Minimalism，暗色主题适配。代码阅读体验优先：降低高对比度色值冲突，统一语法高亮语义。

## 文件改造规格

### codeView/index.css

| 项 | 改造前 | 改造后 |
|----|--------|--------|
| 图片预览过渡 | `transform 0.2s ease, box-shadow 0.2s ease` | `--yi-duration-fast` / `--yi-easing-default` |
| 图片预览阴影 | `0 4px 12px rgba(0,0,0,0.3)` | `--yi-shadow-md` |
| 图片预览 hover 阴影 | `0 8px 24px rgba(0,0,0,0.4)` | `--yi-shadow-lg` |

### codePage.css

| 项 | 改造前 | 改造后 |
|----|--------|--------|
| Fallback 色值 | `var(--border-color, #e9ecef)` | `var(--border-primary)` |
| Fallback 背景 | `var(--bg-hover, #f8f9fa)` | `var(--bg-surface-hover)` |
| 聊天背景渐变 | `linear-gradient(180deg, #0f172a, #1e293b)` | `var(--bg-secondary)` |
| 气泡阴影 | `0 8px 24px rgba(0,0,0,0.18)` | `--yi-shadow-md` |
| 输入框顶部阴影 | `0 -6px 24px rgba(0,0,0,0.28)` | `--yi-shadow-lg` |
| Switch 过渡 | `0.12s ease` | `--yi-duration-fast` |
| 重复定义 | `.code-view-container .empty-title` 定义两次 | 合并为一次 |

### syntax-highlight.css

| 项 | 改造前 | 改造后 |
|----|--------|--------|
| 关键字 | `#c678dd` | `var(--sh-keyword)` |
| 函数 | `#61afef` | `var(--sh-function)` |
| 字符串 | `#98c379` | `var(--sh-string)` |
| 数字 | `#d19a66` | `var(--sh-number)` |
| 注释 | `#5c6370` | `var(--sh-comment)` |
| 变量 | `#e06c75` | `var(--sh-variable)` |
| 类名 | `#e5c07b` | `var(--sh-class)` |
| 操作符 | `#abb2bf` | `var(--sh-operator)` |
| URL | `#56b6c2` | `var(--sh-url)` |

新增 `cdn/styles/components/syntax-highlight.css` 根级 CSS 变量定义（暗色默认值），`@media (prefers-color-scheme: light)` 提供亮色覆盖。

## 无障碍要求

- `prefers-reduced-motion` 禁用 codeView 图片缩放动画
- `prefers-contrast: high` 增强语法高亮对比度
