# 04-测试用例评审 — YiWeb-codeview-markdown-enhance

## 测试策略

人工验证 + DevTools 检查清单 + 静态审查。

## 测试用例

### TC-01: codeView 图片预览

| 项 | 内容 |
|----|------|
| 目的 | 过渡与阴影使用 `--yi-*` |
| 步骤 | 1. 打开代码视图图片预览<br>2. DevTools Computed 检查 transition 和 box-shadow |
| 预期 | `--yi-duration-fast` / `--yi-shadow-md` |

### TC-02: codePage 残留清理

| 项 | 内容 |
|----|------|
| 目的 | 无 fallback 硬编码色值 |
| 步骤 | 1. `grep` 搜索 `#e9ecef` / `#f8f9fa` / `0.12s ease` |
| 预期 | 无命中 |

### TC-03: 语法高亮令牌

| 项 | 内容 |
|----|------|
| 目的 | syntax-highlight 使用语义变量 |
| 步骤 | 1. 检查 `cdn/styles/components/syntax-highlight.css`<br>2. 确认关键字/字符串/注释等使用 `var(--sh-*)` |
| 预期 | 全部命中语义令牌 |

### TC-04: Reduced Motion

| 项 | 内容 |
|----|------|
| 目的 | codeView 动画可被系统偏好禁用 |
| 步骤 | 1. 开启系统减少动画<br>2. 悬停图片预览 |
| 预期 | 无缩放动画 |

## 门禁标准

- TC-01 至 TC-03 为 P0，必须全部通过
- TC-04 为 P1，允许一次修复
