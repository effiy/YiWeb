# 01-故事任务 — YiWeb-shared-components-redesign

## 需求来源

基于 Story 1 `design-system-foundation` 建立的设计令牌，对共享组件进行视觉升级，符合 UI UX Pro Max 的 SaaS/AI 平台设计规范。

## 需求摘要

使用新的 `--yi-*` 设计令牌，对 `cdn/components/` 下的共享组件进行样式升级：
- 统一为 Soft UI + Minimalism 风格
- 降低视觉噪音（减少重渐变、强发光）
- 提升可访问性（焦点环、对比度、触摸目标）
- 补充缺失组件样式（YiErrorState、YiSelect）

| 维度 | 影响范围 | 证据等级 |
|------|---------|---------|
| YiButton | `cdn/components/common/buttons/YiButton/index.css` | A |
| YiIconButton | `cdn/components/common/buttons/YiIconButton/index.css` | A |
| YiModal | `cdn/components/common/modals/YiModal/index.css` | A |
| YiTag | `cdn/components/common/tags/YiTag/index.css` | A |

| YiSelect | `cdn/components/common/forms/YiSelect/index.css` | A |
| YiEmptyState | `cdn/components/common/feedback/YiEmptyState/index.css` | A |
| YiErrorState | `cdn/components/common/feedback/YiErrorState/index.css` | A |

## 依赖

- **上游**: Story 1 `design-system-foundation`（设计令牌必须已就绪）
- **下游**: Story 3 `aicr-layout-ux`（布局组件消费这些共享组件）

## 优先级

P1 — 依赖 Story 1，必须在布局优化之前完成。

## 验收标准

1. 所有共享组件使用 `--yi-*` 令牌而非旧 `--primary` 等映射
2. 按钮取消重渐变和强发光，改用纯色 + 微阴影
3. 模态框阴影柔和，支持 `prefers-reduced-motion`
4. 加载器取消霓虹发光效果，保持简洁
5. YiErrorState、YiSelect 样式补全
6. 所有交互元素最小触摸目标 44×44px
