# 03-前端技术评审 — YiWeb-shared-components-redesign

## 设计方向

基于 UI UX Pro Max 对 SaaS / AI Platform 的推荐：
- **Style**: Minimalism & Swiss Style + Soft UI Evolution
- **Effects**: Soft shadows, subtle depth, smooth transitions (150–300ms)
- **Avoid**: Heavy gradients, neon glows, decorative-only animations

## 组件改造规格

### YiButton

| 项 | 改造前 | 改造后 |
|----|--------|--------|
| Primary | 紫粉渐变 + 强发光 | `--yi-primary` 纯色 + `--yi-shadow-md` |
| Secondary | 玻璃态 + 重阴影 | `--yi-surface` + `--yi-border` |
| Outline | 渐变伪元素 | 透明背景 + 实线边框 |
| Ghost | 同前 | 简化 hover 背景 |
| Hover | translateY(-2px) + 大阴影 | translateY(-1px) + `--yi-shadow-md` |
| Focus | 旧 alpha 环 | `--yi-shadow-focus` |
| 动画 | 0.3s | `--yi-duration-fast` / `--yi-duration-normal` |

### YiModal

| 项 | 改造前 | 改造后 |
|----|--------|--------|
| Overlay | rgba(0,0,0,0.7) | rgba(15,23,42,0.6) + backdrop-blur |
| Background | `--bg-secondary` | `--yi-code-bg` |
| Border | 旧 border-primary | `--yi-border` |
| Shadow | 旧重阴影 | `--yi-shadow-xl` |
| Header padding | 16px 20px | `--yi-space-4` `--yi-space-5` |
| Animation | 0.2s ease | `var(--yi-duration-normal)` + 支持 `prefers-reduced-motion` |

### YiTag

| 项 | 改造前 | 改造后 |
|----|--------|--------|
| 变量 | 旧 `--tag-*` / `--text-secondary` | 直接 `--yi-*` |
| Focus | 旧 `--primary-alpha` | `--yi-shadow-focus` |
| 变体色 | 硬编码 rgba | 使用 `--yi-primary-subtle` 等语义色 |

### YiLoading

| 项 | 改造前 | 改造后 |
|----|--------|--------|
| 双环发光 | 霓虹蓝紫 + 大 glow | 单环 + `--yi-primary` 边框 |
| 阴影 | `0 0 16px 2px ...` | 无发光，纯旋转 |
| 骨架屏 | 旧 bg 色 | `--yi-border-subtle` → `--yi-border` |

### YiEmptyState

| 项 | 改造前 | 改造后 |
|----|--------|--------|
| 颜色 | 旧 `--text-muted` | `--yi-text-muted` |
| Padding | 60px 20px | `--yi-space-12` `--yi-space-5` |
| Icon opacity | 0.5 | 0.4（更柔和） |

### YiErrorState（新增）

- 结构：图标 + 标题 + 描述 + 操作按钮
- 样式：使用 `--yi-danger` + `--yi-danger-subtle` 背景
- 图标：Font Awesome `circle-exclamation`

### YiSelect（新增）

- 结构：触发器 + 下拉面板 + 选项列表
- 样式：`--yi-surface` 背景，`--yi-border` 边框，`--yi-shadow-lg` 下拉阴影
- 选项 hover：`--yi-surface-hover`
- 选中：`--yi-primary-subtle` + `--yi-primary` 文字

## 无障碍要求

- 按钮/选项最小触摸目标 44×44px
- 焦点环可见：`--yi-shadow-focus`
- `prefers-reduced-motion` 禁用所有动画
- 颜色不单独传达信息（配合图标/文字）
