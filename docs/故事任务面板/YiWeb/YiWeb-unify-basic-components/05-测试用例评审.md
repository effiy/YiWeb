# 05 — 测试用例评审：优化统一基础组件大小与配色

## 测试范围

- 组件级：YiInput、YiTextarea、YiButton、YiIconButton、YiTag、YiSelect
- 视图级：聊天输入域、设置弹窗输入、FAQ 搜索输入
- 主题级：暗色 / 亮色 / 高对比 / 减少动效

## 测试策略

零构建项目无自动化测试框架，采用 **手工视觉验证 + 结构化检查清单**。

## 组件级测试用例

## 回归检查清单

- [ ] `YiWeb-unify-theme-colors` 已修复的 legacy 变量无回退
- [ ] YiButton 原有 variant（primary/secondary/outline/ghost/danger）不受影响
- [ ] YiTag 原有 variant（primary/success/warning/danger/info）不受影响
- [ ] YiSelect 原有交互（展开/选择/关闭）不受影响
- [ ] 视图布局无错位（尤其按钮+输入框并排场景）
