## 经验沉淀

1. **组件优先于内联样式**：本次新建 YiInput/YiTextarea 后，后续视图输入替换有明确入口，避免继续扩散内联样式债务。
2. **尺寸 scale 需分族群**：按钮/输入/选择器共享控件 scale（36/44/52），标签因视觉层级不同保持独立 scale（24/28/36），强行统一反而破坏层级。
3. **存量 class 改名需扫引用**：YiTag `.tag-small` → `.tag-sm` 改名前通过 grep 确认无存量引用，避免了静默失效。
