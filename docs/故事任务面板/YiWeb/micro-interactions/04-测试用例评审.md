# 04-测试用例评审 — YiWeb-micro-interactions

## 测试策略

人工验证 + DevTools 检查清单 + 静态审查。

## 测试用例

### TC-01: 离散过渡清理

| 项 | 内容 |
|----|------|
| 目的 | 无 `0.15s` / `0.2s` / `0.3s` 离散时长 |
| 步骤 | `grep` 搜索上述离散值 |
| 预期 | 无命中（`fa-spin` 除外，为 FA 内置） |

### TC-02: 硬编码阴影/背景清理

| 项 | 内容 |
|----|------|
| 目的 | 无 `rgba(255,255,255,0.x)` / `rgba(0,0,0,0.3)` 硬编码 |
| 步骤 | `grep` 搜索上述模式 |
| 预期 | 无命中 |

### TC-03: Hover transform 移除

| 项 | 内容 |
|----|------|
| 目的 | `keyboardShortcutsHelp` 无 `translateX(4px)` |
| 步骤 | 检查 `.shortcut-item:hover` |
| 预期 | 无 transform |

### TC-04: Reduced Motion

| 项 | 内容 |
|----|------|
| 目的 | keyboardShortcutsHelp 含 `prefers-reduced-motion` |
| 步骤 | 检查文件底部媒体查询 |
| 预期 | 存在 |

## 门禁标准

- TC-01 ~ TC-03 为 P0
- TC-04 为 P1
