# 03-前端技术评审 — YiWeb-accessibility-polish

## 设计方向

Soft UI + Minimalism。最后一轮无障碍兜底。

## 文件改造规格

### index.css

| 项 | 改造前 | 改造后 |
|----|--------|--------|
| unsaved 指示器 | `animation: unsavedPulse 2s infinite` | 移除默认动画（关键帧保留，JS 动态触发） |
| action-button hover | `translateY(-2px) scale(1.05)` | `translateY(-1px)` |
| action-button shadow | `0 4px 8px rgba(0,0,0,0.15)` | `--yi-shadow-sm` |
| active-match 阴影 | `0 0 8px rgba(234,179,8,0.5)` | 移除发光阴影 |
| reduced-motion | 缺失 | 新增 |

### fileTreeTags.css

| 项 | 改造前 | 改造后 |
|----|--------|--------|
| reduced-motion | 缺失 | 新增 |

### SearchHeader/index.css

| 项 | 改造前 | 改造后 |
