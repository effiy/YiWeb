# 01-故事任务 — YiWeb-accessibility-polish

## 需求来源

基于 Story 1~6 建立的设计令牌，进行最后一轮无障碍补齐，符合 UI UX Pro Max 的 SaaS/AI 平台设计规范。

## 需求摘要

- 为所有缺少 `prefers-reduced-motion` 的 CSS 文件补充媒体查询
- 清理残留的硬编码色值与 legacy 变量
- 简化过度活跃的 hover transform（降低视觉噪音）

## 影响分析

| 维度 | 影响范围 | 证据等级 |
|------|---------|---------|
| index.css | `src/views/aicr/styles/index.css` | A |
| fileTreeTags | `src/views/aicr/components/fileTree/fileTreeTags.css` | A |
| SearchHeader | `cdn/components/business/SearchHeader/index.css` | A |
| YiErrorState | `cdn/components/common/feedback/YiErrorState/index.css` | B |

## 依赖

- **上游**: Story 1~6（设计令牌已就绪）
- **下游**: 无

## 优先级

P1

## 验收标准

1. 所有含 transition/animation/transform 的 CSS 文件含 `prefers-reduced-motion`
2. 无 `rgba(234,179,8,1)` 等硬编码语义色
3. 无 `var(--shadow-sm)` / `var(--transition-fast)` 等 legacy 变量残留
