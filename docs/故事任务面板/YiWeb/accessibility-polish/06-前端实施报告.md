# 06-前端实施报告 — YiWeb-accessibility-polish

## 实施概览

| 项 | 值 |
|----|-----|
| 故事名 | accessibility-polish |
| 分支 | feat/YiWeb-accessibility-polish |
| 实施范围 | 4 个 CSS 文件 |
| 风格 | Soft UI + Minimalism |

## 文件改造详情

### index.css

- `.unsaved-indicator`: 移除默认触发的 `animation: unsavedPulse`（关键帧保留，由 JS 动态添加类触发）
- `.action-button:hover`: `translateY(-2px) scale(1.05)` → `translateY(-1px)`
- `.action-button:hover`: `box-shadow: 0 4px 8px rgba(0,0,0,0.15)` → `var(--yi-shadow-sm)`
- `.code-line.active-match::before`: 移除 `box-shadow: 0 0 8px rgba(234,179,8,0.5)` 发光阴影
- 新增 `@media (prefers-reduced-motion: reduce)` 覆盖 sidebar/action-button/line-copy-btn

### fileTreeTags.css

- 新增 `@media (prefers-reduced-motion: reduce)` 覆盖 tag-filter-btn/tags-clear-btn/tag-item

### SearchHeader/index.css

- `var(--shadow-sm)` → `var(--yi-shadow-sm)`
- `var(--transition-fast)` → `var(--yi-duration-fast) var(--yi-easing-default)`
- `var(--transition-normal)` → `var(--yi-duration-normal) var(--yi-easing-default)`
- `var(--shadow-primary)` → `var(--yi-shadow-md)`
- focus shadow: `0 0 0 3px var(--primary-alpha), var(--shadow-primary)` → `var(--yi-shadow-focus)`
- 新增 `@media (prefers-reduced-motion: reduce)`

### YiErrorState/index.css

- `border: 1px solid rgba(239, 68, 68, 0.15)` → `rgba(var(--yi-danger-rgb), 0.15)`

## 令牌覆盖

- 阴影全面使用 `--yi-shadow-*`
- 过渡全面使用 `--yi-duration-*` + `--yi-easing-default`

## 可访问性

- `prefers-reduced-motion` 已覆盖所有含 transition/animation 的组件
- 过度活跃的 hover transform 已降低幅度或移除
