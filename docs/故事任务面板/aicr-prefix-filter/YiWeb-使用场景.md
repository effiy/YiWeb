### §0 基线声明

> **用户空间基线**: 本文档定义"谁使用(WHO)"和"如何体验(HOW)"。禁止出现技术术语、组件名、API 端点、文件路径。

### 主要价值

- 👤 用户视角完整 — 覆盖过滤操作的入口、执行、撤销全流程

- ⚡ 即时反馈 — 点击标签后文件列表即时更新，搜索体验流畅

## §1 场景

### 场景 1: 按文件名前缀过滤文件（正常路径）

**参与者**: 代码审查者
**前置条件**: 已打开代码审查页面，文件树已加载，包含多种前缀命名的文件（如 `use-auth.js`、`use-store.js`、`api-client.js`、`api-server.js`、`index.html`、`index.css`）

```mermaid
flowchart LR
    A["浏览文件树"] --> B["看到前缀标签区<br/>use(2) api(2) index(2)"]
    B --> C["点击 use 标签"]
    C --> D["文件树仅显示<br/>use-auth.js, use-store.js"]
    D --> E["use 标签高亮"]
    E --> F["继续浏览或切换筛选"]
```

| 步骤 | 用户操作 | 系统行为 |
|------|---------|---------|
| 1 | 查看筛选栏 | 在目录标签下方显示前缀标签区，标签如"use (2)"、"api (2)"、"index (2)"，按数量降序排列 |
| 2 | 点击"use"标签 | 标签高亮为选中态，文件树立即更新，仅显示文件名以"use"开头的文件 |
| 3 | 观察文件列表 | 其他前缀的文件（api-*、index.*）不再显示 |

### 场景 2: 组合使用目录标签和前缀标签（高级路径）

**参与者**: 代码审查者
**前置条件**: 已按场景 1 操作或在任意筛选状态

```mermaid
flowchart LR
    A["已选择目录标签 src"] --> B["文件树显示 src 目录下所有文件"]
    B --> C["点击前缀标签 use"]
    C --> D["文件树仅显示<br/>src/ 下 use- 前缀的文件"]
    D --> E["两个维度标签<br/>同时高亮"]
```

| 步骤 | 用户操作 | 系统行为 |
|------|---------|---------|
| 1 | 在顶部标签栏选中"src"目录标签 | 文件树过滤为 src 目录下的内容 |
| 2 | 在筛选栏点击"use"前缀标签 | 文件树进一步收缩，仅保留 src 目录中以 use 开头的文件 |
| 3 | 取消"src"标签 | 文件树显示所有目录中以 use 开头的文件，前缀标签保持选中 |

### 场景 3: 清除前缀筛选（撤销路径）

**参与者**: 代码审查者
**前置条件**: 已选中一个或多个前缀标签

```mermaid
flowchart LR
    A["已选中 use 前缀标签"] --> B["点击清除按钮"]
    B --> C["所有前缀标签取消选中"]
    C --> D["文件树恢复仅目录标签筛选"]
```

| 步骤 | 用户操作 | 系统行为 |
|------|---------|---------|
| 1 | 查看已选中的前缀标签 | 一个或多个前缀标签处于高亮状态 |
| 2 | 点击前缀标签区的"清除"按钮 | 所有前缀标签取消选中，文件树恢复为无前缀筛选的状态 |
| 3 | 或点击已高亮的前缀标签 | 仅该标签取消选中，其他标签保持 |

### 场景 4: 空状态 — 无可提取前缀（边界路径）

**参与者**: 代码审查者
**前置条件**: 文件树中所有文件名均无可识别前缀（如文件名为 `a`、`b`、`c` 或全无分隔符）

| 步骤 | 用户操作 | 系统行为 |
|------|---------|---------|
| 1 | 打开页面，查看筛选栏 | 前缀标签区域整体不显示，仅显示目录标签区 |
| 2 | 正常使用目录标签筛选 | 不受影响 |

### 场景 5: 选中所有文件都匹配的前缀后切换视图（边界路径）

**参与者**: 代码审查者
**前置条件**: 所有文件共享同一前缀（如项目所有文件都以 `app-` 开头）

| 步骤 | 用户操作 | 系统行为 |
|------|---------|---------|
| 1 | 查看前缀标签 | 显示"app (N)"标签（N = 文件总数） |
| 2 | 点击"app"标签 | 所有文件仍显示（因全部匹配），标签高亮 |
| 3 | 取消选中 | 恢复，无变化 |

## §3 范围边界

**范围内**: 前缀自动提取 · 前缀标签展示与切换 · 与目录标签组合筛选 · 清除功能
**范围外**: 手动创建前缀规则 · 正则表达式自定义 · 前缀标签拖拽排序 · 持久化前缀选中状态
