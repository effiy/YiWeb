<a id="sec1-scope"></a>

## §1 测试范围

| 维度 | 范围 |
|------|------|
| 前缀提取 | 正确性、边界（无分隔符、多分隔符、特殊字符） |
| 前缀标签 UI | 渲染、计数、排序、空状态隐藏 |
| 前缀筛选 | 点击切换、多选、清除、与目录标签组合 |
| 回归 | 现有目录标签筛选不受影响 |

**不测**: 前缀标签拖拽排序（范围外）、正则匹配（范围外）、持久化（范围外）

<a id="sec2-cases"></a>

## §2 用例

### TC-N-01: 前缀标签正常渲染
| 字段 | 内容 |
|------|------|
| 优先级 | P0 |
| 类型 | 正常路径 |
| Given | 文件树已加载，包含文件: `use-auth.js`、`use-store.js`、`api-client.js`、`api-server.js`、`index.html` |
| When | 页面渲染完成 |
| Then | 筛选栏显示前缀标签区，含标签 "use (2)"、"api (2)"、"index (1)"，按数量降序排列 |

### TC-N-02: 点击前缀标签筛选文件
| 字段 | 内容 |
|------|------|
| 优先级 | P0 |
| 类型 | 正常路径 |
| Given | 文件树可见，前缀标签 "use (2)" 和 "api (2)" 显示 |
| When | 点击 "use" 标签 |
| Then | 文件树仅显示 `use-auth.js`、`use-store.js`；"use" 标签高亮；再次点击该标签，取消选中，文件树恢复全部显示 |

### TC-N-03: 目录标签 + 前缀标签组合筛选
| 字段 | 内容 |
|------|------|
| 优先级 | P0 |
| 类型 | 正常路径 |
| Given | 文件树含 src 目录（下有 `use-auth.js`、`api-client.js`）和 lib 目录（下有 `use-store.js`） |
| When | 先选中目录标签 "src"，再点击前缀标签 "use" |
| Then | 文件树仅显示 src 目录下的 `use-auth.js`；取消目录标签后前缀标签保持选中，显示所有目录的 use- 前缀文件 |

### TC-N-04: 多前缀标签 OR 筛选
| 字段 | 内容 |
|------|------|
| 优先级 | P1 |
| 类型 | 正常路径 |
| Given | 前缀标签 "use (2)"、"api (2)"、"index (1)" 可见 |
| When | 依次点击 "use" 和 "api" |
| Then | 两个标签均高亮，文件树同时显示 use- 和 api- 前缀的文件（OR 逻辑），index.* 文件不显示 |

### TC-B-01: 无前缀文件空状态
| 字段 | 内容 |
|------|------|
| 优先级 | P0 |
| 类型 | 边界 |
| Given | 文件树中所有文件名均无 `-`、`_`、`.` 分隔符（如 `a`、`b`、`c`） |
| When | 页面渲染完成 |
| Then | 前缀标签区域整体不显示 |

### TC-B-02: 单文件无有效前缀
| 字段 | 内容 |
|------|------|
| 优先级 | P1 |
| 类型 | 边界 |
| Given | 文件树含 `readme`（无分隔符）和 `src/main.js`（有分隔符） |
| When | 前缀提取执行 |
| Then | 仅 "main (1)" 显示为前缀标签，"readme" 被忽略 |

### TC-E-01: 清除前缀筛选
| 字段 | 内容 |
|------|------|
| 优先级 | P0 |
| 类型 | 异常恢复 |
| Given | 已选中 "use" 和 "api" 两个前缀标签 |
| When | 点击前缀标签区的清除按钮 |
| Then | 所有前缀标签取消选中，文件树恢复无前缀筛选状态 |

### TC-R-01: 现有目录标签筛选不受影响
| 字段 | 内容 |
|------|------|
| 优先级 | P0 |
| 类型 | 回归 |
| Given | 页面正常加载，前缀标签功能就绪 |
| When | 仅使用目录标签（一级/二级）进行筛选 |
| Then | 筛选行为与加入前缀功能前完全一致 |

## §3 Gate A 交接

| 阻断用例 | 验证命令 | 预期结果 |
|---------|---------|---------|
| TC-N-01 | 浏览器中加载含多前缀文件的文件树 | 前缀标签显示，计数正确 |
| TC-N-02 | 浏览器中点击前缀标签 | 文件树过滤正确，标签高亮切换正常 |
| TC-N-03 | 浏览器中组合使用目录+前缀标签 | 组合筛选结果正确（AND 逻辑） |
| TC-B-01 | 浏览器中加载全无分隔符文件名的文件树 | 前缀标签区不显示 |

**Gate A 通过条件**: TC-N-01、TC-N-02、TC-B-01 全部通过。
