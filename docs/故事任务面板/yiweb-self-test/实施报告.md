## §7 可操作验证

### 实施步骤

1. 创建 `package.json`（如不存在）并安装 Vitest：
   ```bash
   npm init -y && npm install -D vitest jsdom @vitest/coverage-v8
   ```
2. 创建 `vitest.config.js`（按技术评审 §2 配置）
3. 创建 `tests/setup.js`（按技术评审 §6 全局 mock）
4. 按优先级创建测试文件（P0 先于 P1）：
   - `tests/cdn/utils/core/log.test.js`
   - `tests/cdn/utils/core/error.test.js`
   - `tests/src/core/services/helper/authUtils.test.js`
   - ...（共 11 个文件）
5. 运行全量测试：`npx vitest run`
6. 运行覆盖率：`npx vitest run --coverage`

### 快速自检

```bash
# 框架就绪检查
npx vitest --version  # 预期输出 Vitest 版本号

# 运行测试
npx vitest run  # 预期全部通过或列出失败项

# 覆盖率
npx vitest run --coverage  # 预期输出 L1/L2 覆盖率报告
```
