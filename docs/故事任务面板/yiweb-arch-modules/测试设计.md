### 主要价值

- ✅ 验证模块入口全覆盖 — 五大分类的模块入口文件均存在

- 🚦 确认依赖方向无违规 — CDN 模块不依赖 src 模块
- 📊 保证模块地图总表达标 — ≥ 20 个模块有完整三元组

## §1 测试范围

| 维度 | 覆盖 | 不覆盖（原因） |
|------|------|---------------|
| 模块存在性 | 五大分类全部模块入口文件 | 外部 CDN 依赖（非本仓库管理） |
| 三元组完整性 | 入口 + 内部依赖 + 下游消费者三列 | 下游消费者的运行时动态 import |
| 依赖方向 | CDN→src 违规检查 | 间接依赖（事件总线等） |
| 分类正确性 | 按判定规则验证模块分类 | 分类边界模糊的模块（需人工判定） |
