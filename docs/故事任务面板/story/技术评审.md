## §2 数据流

```mermaid
flowchart LR
    MOUNT["onMounted"] --> FETCH["store.fetchStories()"]
    FETCH --> API["远端 API 查询"]
    API --> STATE["store.stories[]"]
    STATE --> COMPUTED["computed 过滤/排序"]
    COMPUTED --> RENDER["列表/卡片/详情渲染"]
    FILTER["搜索/筛选"] --> COMPUTED
    SORT["排序切换"] --> COMPUTED
    CLICK["点击故事"] --> SELECT["store.selectStory(name)"]
    SELECT --> DETAIL["StoryDetailCard 渲染"]
```

> 证据: `src/views/story/index.js` · `src/views/story/hooks/store.js`

## §4 状态判定逻辑

```
故事阶段 = function(文档存在性):
  hasStoryTask  + hasScenario + hasTechReview + hasSecurityAudit + hasTestDesign
  5/5 → completed
  3-4/5 → in_progress
  1-2/5 → pending (doc stage)
  0/5 → pending (init)
```
