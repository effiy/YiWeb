### 主要价值

- 🎯 只读视图安全面小 — 无用户输入写入，仅查询远端 API
- 🔒 XSS 防御 — 故事标题/描述经框架自动转义
- ⚡ STRIDE 全覆盖 — 六类威胁逐项建模

---

## §1 STRIDE 威胁建模

### S — Spoofing | T — Tampering

| 威胁 | 缓解 |
|------|------|
| 伪造故事数据 | 数据来自认证 API，客户端不可篡改 |
| 篡改状态判定 | 状态判定在浏览器端执行，仅用于展示 |

### I — Information Disclosure

| 威胁 | 缓解 |
|------|------|
| 故事标题含恶意脚本 | 模板引擎自动转义 |
| API 响应泄漏敏感路径 | 仅展示文件名和元数据，不含文件内容 |

### D — Denial of Service

| 威胁 | 缓解 |
|------|------|
| 大量故事导致页面卡顿 | 并发类型推断限制 4 并发 |

### E — Elevation of Privilege

| 威胁 | 缓解 |
|------|------|
| 通过面板修改故事状态 | 面板仅展示，无写入操作 |
