## §1 STRIDE 威胁建模

### S — Spoofing | T — Tampering

| 威胁 | 缓解 |
|------|------|
| 伪造故事数据 | 数据来自认证 API |
| 通过面板修改故事 | 面板无写入操作 |

### I — Information Disclosure

| 威胁 | 缓解 |
|------|------|
| 故事信息泄漏 | 数据经 API 认证返回 |
| 搜索关键词通过 URL 暴露 | 搜索仅前端过滤，不发送到 URL |

### D — Denial of Service

| 威胁 | 缓解 |
|------|------|
| 大量故事导致列表性能下降 | computed 缓存过滤结果 |

### E — Elevation of Privilege

| 威胁 | 缓解 |
|------|------|
| 通过面板操作修改故事 | 面板仅展示，无操作接口 |
