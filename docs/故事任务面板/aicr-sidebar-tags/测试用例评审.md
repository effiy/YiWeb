# 测试用例评审：aicr-sidebar-tags

## 评审结论

纯前端 UI 改动，测试范围：标签 UI 渲染、交互事件、侧边栏 sticky 行为、响应式兼容。

## 测试用例清单

### 冒烟测试（P0）

| # | 用例 | 前置条件 | 操作 | 预期结果 |
|---|------|---------|------|---------|
| T1 | 标签列表渲染 | 文件树数据含标签 | 打开 AICR 页面 | 搜索框下方显示标签列表，标签数量与 `allTags` 一致 |
| T2 | 标签点击筛选 | 标签列表可见 | 点击标签 "frontend" | 标签高亮，文件树按该标签筛选，header 中同步高亮 |
| T3 | 标签取消选择 | 标签 "frontend" 已选中 | 再次点击 "frontend" | 标签取消高亮，文件树恢复全量 |
| T4 | 侧边栏吸顶 | 页面含足够内容可滚动 | 向下滚动代码区 | 侧边栏保持在视口顶部 |
