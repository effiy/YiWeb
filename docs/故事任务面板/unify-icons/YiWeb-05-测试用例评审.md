## §3 门禁规则

| 等级 | 条件 | 行为 |
|:----:|------|------|
| P0 | iconMap.js 未覆盖某个现有图标 | 补齐后放行 |
| P0 | 视图模板残留 `fas fa-*` 硬编码 | 迁移后放行 |
| P0 | 图标渲染异常（不显示或显示错误图标） | 修复映射后放行 |
| P1 | JS 文件残留 `fas fa-*`（除映射表） | 迁移后放行 |
| P2 | 动态图标绑定未迁移 | 记录不阻断 |
