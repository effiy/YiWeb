## 实施概述

本次实施将 AICR 页面中的会话标签筛选器从侧边栏的 FileTree 组件中移出，创建了独立的 SessionListTags 组件，并放置在页面顶部 header 下方，使标签筛选功能更加醒目和易用。

## 架构变更

### 组件结构变更

**修改前：**
```
AicrPage
├── AicrHeader
├── AicrMain
│   ├── AicrSidebar
│   │   └── FileTree (包含标签筛选器)
│   └── AicrCodeArea
└── ...
```

**修改后：**
```
AicrPage
├── AicrHeader
├── SessionListTags (新组件，独立的标签筛选器)
├── AicrMain
│   ├── AicrSidebar
│   │   └── FileTree (移除标签筛选器)
│   └── AicrCodeArea
└── ...
```

### 数据流变更

- **修改前：** 标签筛选事件由 FileTree 触发
- **修改后：** 标签筛选事件由 SessionListTags 触发
- **保持不变：** Store 中的标签状态结构完全不变
